## 📌 Executive Summary

This lecture provides the complete **C program** for Circular Linked List — creation of nodes, linking them in a circular fashion, and displaying using a do-while loop.

---

## 💻 Complete Program
```c
#include <stdio.h>
#include <stdlib.h>

struct node {
    int data;
    struct node *next;
};
struct node *head = NULL;

void create(int n) {
    struct node *newNode, *temp;
    for (int i = 0; i < n; i++) {
        newNode = (struct node *)malloc(sizeof(struct node));
        printf("Enter data: ");
        scanf("%d", &newNode->data);

        if (head == NULL) {
            head = newNode;
            newNode->next = head;
        } else {
            temp = head;
            while (temp->next != head)
                temp = temp->next;
            temp->next = newNode;
            newNode->next = head;
        }
    }
}

void display() {
    if (head == NULL) { printf("List empty\n"); return; }
    struct node *temp = head;
    do {
        printf("%d → ", temp->data);
        temp = temp->next;
    } while (temp != head);
    printf("(back to head)\n");
}

int main() {
    int n;
    printf("Enter number of nodes: ");
    scanf("%d", &n);
    create(n);
    display();
    return 0;
}
```

---

## 💡 Key Takeaways

- Same struct as SLL — difference is in linking logic
- Last node always links to `head`
- Use `while (temp->next != head)` to traverse to last node
- Display uses `do-while` with condition `temp != head`
