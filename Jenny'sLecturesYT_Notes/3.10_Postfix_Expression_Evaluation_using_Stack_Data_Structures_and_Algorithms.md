## 📐 Example: Evaluate `2 3 1 * + 9 -`

| Symbol | Action | Stack |
|--------|--------|-------|
| 2 | push | [2] |
| 3 | push | [2, 3] |
| 1 | push | [2, 3, 1] |
| * | pop 1,3 → 3×1=3 | [2, 3] |
| + | pop 3,2 → 2+3=5 | [5] |
| 9 | push | [5, 9] |
| - | pop 9,5 → 5-9=-4 | [-4] |

**Result = -4** ✅

## 💻 C Code
```c
int evaluatePostfix(char *exp) {
    int stack[100], top = -1;
    for (int i = 0; exp[i]; i++) {
        if (isdigit(exp[i]))
            stack[++top] = exp[i] - '0';
        else {
            int op2 = stack[top--];
            int op1 = stack[top--];
            switch(exp[i]) {
                case '+': stack[++top] = op1 + op2; break;
                case '-': stack[++top] = op1 - op2; break;
                case '*': stack[++top] = op1 * op2; break;
                case '/': stack[++top] = op1 / op2; break;
            }
        }
    }
    return stack[top];
}
```
