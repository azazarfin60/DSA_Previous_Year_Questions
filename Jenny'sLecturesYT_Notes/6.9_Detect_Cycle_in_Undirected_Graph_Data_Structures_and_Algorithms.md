## 📐 Method: DFS with Parent Tracking
> In an undirected graph, a cycle exists if DFS finds a visited vertex that is **NOT the parent** of the current vertex.

```c
int hasCycleDFS(int v, int parent, int visited[]) {
    visited[v] = 1;
    for each neighbor u of v:
        if (!visited[u]) {
            if (hasCycleDFS(u, v, visited)) return 1;
        } else if (u != parent) {
            return 1;  // Visited non-parent → CYCLE!
        }
    return 0;
}
```

## ⚠️ Why parent check?
> In undirected graph, edge A-B means B is a neighbor of A. When DFS goes A→B, then checks B's neighbors, A will appear. Without parent check, this would falsely detect a cycle.

## 📐 Alternative: Union-Find
```
For each edge (u, v):
    if find(u) == find(v) → cycle!
    else union(u, v)
```

## ⏱️ Time: O(V + E)
