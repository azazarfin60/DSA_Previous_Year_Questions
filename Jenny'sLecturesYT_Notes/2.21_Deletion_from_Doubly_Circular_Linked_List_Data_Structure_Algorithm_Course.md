## 💻 Delete from Beginning
```c
void deleteFromBeginning() {
    if (head == NULL) return;
    if (head->next == head) { free(head); head = NULL; return; }
    struct node *last = head->prev;  // O(1)!
    struct node *temp = head;
    head = head->next;
    head->prev = last;
    last->next = head;
    free(temp);
}
```
**Time: O(1)** — thanks to `head->prev` giving last node

## 💻 Delete from End
```c
void deleteFromEnd() {
    if (head == NULL) return;
    if (head->next == head) { free(head); head = NULL; return; }
    struct node *last = head->prev;
    last->prev->next = head;
    head->prev = last->prev;
    free(last);
}
```
**Time: O(1)** — no traversal needed!

## 💡 Key: DCLL gives O(1) deletion from both ends.
