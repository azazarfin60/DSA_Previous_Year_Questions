## 🔑 Definition
> An Expression Tree is a binary tree where **leaves are operands** and **internal nodes are operators**. Traversals give different expression notations.

```
       *
      / \
     +   -
    / \ / \
   A  B C  D

Inorder:   A + B * C - D  (Infix)
Preorder:  * + A B - C D  (Prefix)
Postorder: A B + C D - *  (Postfix)
```

## 💡 Key: Inorder = Infix, Preorder = Prefix, Postorder = Postfix
