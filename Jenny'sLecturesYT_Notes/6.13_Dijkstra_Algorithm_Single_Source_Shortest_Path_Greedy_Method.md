# 6.13 Dijkstra Algorithm | Single Source Shortest Path | Greedy Method

📺 [Watch on YouTube](https://www.youtube.com/watch?v=smHnz2RHJBY)

---

## 📌 Executive Summary

Dijkstra's algorithm finds the **shortest path from a single source** to all other vertices in a weighted graph with **non-negative edges**. Uses a greedy approach — always picks the closest unvisited vertex.

---

## 📐 Algorithm

1. Set `dist[source] = 0`, `dist[all others] = ∞`
2. While unvisited vertices exist:
   - Pick vertex **u** with minimum `dist[u]` from unvisited
   - Mark u as visited
   - For each neighbor v of u:
     - If `dist[u] + weight(u,v) < dist[v]`:
       - `dist[v] = dist[u] + weight(u,v)` (relaxation)

---

## 📐 Example

```
Graph:
  A --4-- B
  |      /|
  1    2  6
  |  /    |
  C --3-- D --2-- E
```

### Dijkstra from A:

| Step | Visited | dist[A] | dist[B] | dist[C] | dist[D] | dist[E] | Pick |
|------|---------|---------|---------|---------|---------|---------|------|
| Init | {} | 0 | ∞ | ∞ | ∞ | ∞ | A(0) |
| 1 | {A} | 0 | 4 | 1 | ∞ | ∞ | C(1) |
| 2 | {A,C} | 0 | 3 | 1 | 4 | ∞ | B(3) |
| 3 | {A,C,B} | 0 | 3 | 1 | 4 | ∞ | D(4) |
| 4 | {A,C,B,D} | 0 | 3 | 1 | 4 | 6 | E(6) |

### Shortest Distances from A:
```
A → A = 0
A → B = 3  (via A→C→B)
A → C = 1  (direct)
A → D = 4  (via A→C→D)
A → E = 6  (via A→C→D→E)
```

---

## ⚠️ Limitations
- **Cannot handle negative weights** → use Bellman-Ford instead
- Greedy choice: once a vertex is finalized, distance won't change (fails with negative edges)

---

## ⏱️ Complexity

| Implementation | Time |
|---------------|------|
| Adjacency Matrix | **O(V²)** |
| Min Heap + Adj List | **O((V+E) log V)** |

---

## 📊 Dijkstra vs Bellman-Ford vs Floyd-Warshall

| Algorithm | Type | Negative Edges | Time |
|-----------|------|---------------|------|
| **Dijkstra** | Single source | ❌ No | O(V²) |
| **Bellman-Ford** | Single source | ✅ Yes | O(VE) |
| **Floyd-Warshall** | All pairs | ✅ Yes | O(V³) |

---

## 💡 Key Takeaways

- **Greedy**: always pick closest unvisited vertex
- **Relaxation**: update dist if shorter path found
- Cannot handle **negative weights**
- Draw the **table** step-by-step in exams for full marks
- Most frequently asked graph algorithm!
