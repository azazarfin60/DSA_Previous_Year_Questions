## 📐 Detailed Trace Example
```
         A
        / \
       B   C
      / \   \
     D   E   F

Preorder (NLR): A, B, D, E, C, F
Inorder  (LNR): D, B, E, A, C, F
Postorder(LRN): D, E, B, F, C, A
```

## 📐 Trace: Inorder step-by-step
```
inorder(A) → go left → inorder(B) → go left → inorder(D)
  → D.left=NULL → print D → D.right=NULL → return
→ print B → inorder(E) → E.left=NULL → print E → return
→ print A → inorder(C) → C.left=NULL → print C → inorder(F)
  → F.left=NULL → print F → return

Result: D, B, E, A, C, F ✅
```
