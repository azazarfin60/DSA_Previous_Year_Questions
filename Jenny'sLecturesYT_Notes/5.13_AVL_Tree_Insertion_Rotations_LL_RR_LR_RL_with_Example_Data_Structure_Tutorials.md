## 🔑 AVL = Self-balancing BST with Balance Factor (BF) ∈ {-1, 0, 1}
> BF(node) = Height(left subtree) - Height(right subtree)

## 📐 Four Rotation Cases

### LL (Left-Left) — Right Rotate
```
      z                y
     / \             /   \
    y   T4   →      x     z
   / \             / \   / \
  x   T3         T1  T2 T3  T4
 / \
T1  T2
```

### RR (Right-Right) — Left Rotate
```
  z                    y
 / \                 /   \
T1   y       →      z     x
    / \            / \   / \
   T2   x        T1  T2 T3  T4
       / \
      T3  T4
```

### LR (Left-Right) — Left Rotate then Right Rotate
```
    z              z              x
   / \            / \           /   \
  y   T4  →     x   T4  →    y     z
 / \           / \           / \   / \
T1   x        y   T3       T1  T2 T3  T4
    / \      / \
   T2  T3  T1  T2
```

### RL (Right-Left) — Right Rotate then Left Rotate
```
  z              z                  x
 / \            / \               /   \
T1   y   →    T1   x      →    z     y
    / \           / \          / \   / \
   x   T4       T2   y      T1  T2 T3  T4
  / \               / \
 T2  T3            T3  T4
```

## 📐 Example: Insert 10, 20, 30, 40, 50, 25

```
Insert 10:    10

Insert 20:    10
                \
                20

Insert 30:    BF(10)=-2, RR → Left Rotate on 10
              20
             /  \
           10   30

Insert 40:    BF(20)=-1, OK. BF(30)=-1, OK.
              20
             /  \
           10   30
                  \
                  40

Insert 50:    BF(30)=-2, RR → Left Rotate on 30
              20
             /  \
           10   40
               /  \
             30   50

Insert 25:    BF(20)=-2, RL → Right Rotate on 40, then Left Rotate on 20
              30
             /  \
           20    40
          /  \     \
        10   25    50
```

## ⏱️ All operations: **O(log n)** guaranteed.
