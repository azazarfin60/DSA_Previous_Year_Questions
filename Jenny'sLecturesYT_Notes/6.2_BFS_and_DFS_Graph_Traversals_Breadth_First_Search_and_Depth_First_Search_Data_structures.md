## 📌 Executive Summary

This lecture covers the two fundamental graph traversal algorithms: **BFS** (using Queue) and **DFS** (using Stack/Recursion), with step-by-step traces.

---

## 📐 BFS (Breadth First Search)

### Algorithm
1. Start from source vertex, enqueue it, mark visited
2. While queue not empty:
   - Dequeue a vertex
   - Visit all **unvisited neighbors**, enqueue and mark them

### Uses: **Queue**

### Example
```
Graph:          BFS from 0:
  0 — 1         Queue: [0] → visit 0
  |   |         Queue: [1, 3] → visit 1
  3 — 2         Queue: [3, 2] → visit 3
                Queue: [2] → visit 2

BFS Order: 0, 1, 3, 2
```

### Pseudocode
```c
void BFS(int start) {
    int visited[MAX] = {0};
    enqueue(start);
    visited[start] = 1;

    while (!isEmpty()) {
        int v = dequeue();
        printf("%d ", v);

        for each neighbor u of v:
            if (!visited[u]) {
                visited[u] = 1;
                enqueue(u);
            }
    }
}
```

## 📐 DFS (Depth First Search)

### Algorithm
1. Start from source, push to stack, mark visited
2. While stack not empty:
   - Pop a vertex, visit it
   - Push all **unvisited neighbors** to stack

### Uses: **Stack** (or Recursion)

### Example
```
Graph:          DFS from 0:
  0 — 1         Stack: [0] → pop 0, push 1,3
  |   |         Stack: [1, 3] → pop 3, push 2
  3 — 2         Stack: [1, 2] → pop 2
                Stack: [1] → pop 1

DFS Order: 0, 3, 2, 1 (depends on adjacency order)
```

### Recursive DFS
```c
void DFS(int v, int visited[]) {
    visited[v] = 1;
    printf("%d ", v);

    for each neighbor u of v:
        if (!visited[u])
            DFS(u, visited);
}
```

## 📊 BFS vs DFS

| Feature | BFS | DFS |
|---------|-----|-----|
| Data structure | **Queue** | **Stack / Recursion** |
| Strategy | Level by level | Depth first |
| Shortest path (unweighted) | ✅ **Yes** | ❌ No |
| Space | O(V) | O(V) |
| Time | **O(V + E)** | **O(V + E)** |
| Complete? | Yes | Yes (finite graph) |
| Use case | Shortest path, level order | Topological sort, cycle detect |

---

## ⏱️ Complexity (Both)

| Representation | Time |
|---------------|------|
| Adjacency List | **O(V + E)** |
| Adjacency Matrix | **O(V²)** |

---

## 💡 Key Takeaways

- BFS = Queue = **level-by-level** = shortest path
- DFS = Stack/Recursion = **go deep first**
- Both visit every vertex exactly once → O(V + E)
- BFS is better for **shortest path**, DFS for **cycle detection**
- Both need a **visited array** to avoid infinite loops
