## 💻 C Code
```c
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++)
            if (arr[j] < arr[minIdx])
                minIdx = j;
        // Swap arr[i] and arr[minIdx]
        int temp = arr[i]; arr[i] = arr[minIdx]; arr[minIdx] = temp;
    }
}
```

## ⏱️ Always O(n²) (all cases) | Stable: ❌ No | In-place: ✅ | Minimum swaps: O(n)
