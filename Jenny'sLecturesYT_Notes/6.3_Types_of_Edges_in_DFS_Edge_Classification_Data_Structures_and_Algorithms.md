# 6.3 Types of Edges in DFS | Edge Classification | Data Structures and Algorithms

## 📌 Executive Summary

During DFS traversal, edges are classified into **four types**: Tree edges, Back edges, Forward edges, and Cross edges. Back edges indicate **cycles**.

---

## 📐 Four Edge Types

| Edge Type | Description | Condition |
|-----------|-------------|-----------|
| **Tree Edge** | Edge in the DFS tree | Discovery edge to unvisited vertex |
| **Back Edge** | Edge to an **ancestor** | Points to a vertex on the current path (in stack) |
| **Forward Edge** | Edge to a **descendant** (non-tree) | Points to already visited vertex in same subtree |
| **Cross Edge** | Edge to a vertex in **different subtree** | Points to already visited vertex in different subtree |

---

## 📐 Identification Using Discovery/Finish Times

Each vertex gets:
- **d[v]** = discovery time (when first visited)
- **f[v]** = finish time (when all descendants processed)

```
Edge (u, v):
Tree Edge:    d[u] < d[v] and v is first discovered via u
Back Edge:    d[u] > d[v] and f[v] not yet set (v is ancestor)
Forward Edge: d[u] < d[v] and v already finished
Cross Edge:   d[u] > d[v] and v already finished
```

---

## 📐 Example
```
Graph: A→B, A→C, B→D, D→C, D→A

DFS from A:
A(1) → B(2) → D(3) → C(4)
                ↓ D→A is Back Edge (A is ancestor of D)
                ↓ D→C is Cross/Forward Edge

Edge Classification:
A→B : Tree Edge
B→D : Tree Edge
D→C : Cross Edge (C in different subtree initially)
D→A : Back Edge (A is ancestor) → CYCLE EXISTS!
```

---

## 🎯 Key Application: Cycle Detection

> **A directed graph has a cycle ⟺ DFS produces a Back Edge**

| Graph Type | Edges Present |
|-----------|---------------|
| Undirected | Tree + Back only |
| Directed (DAG) | Tree + Forward + Cross (NO Back) |
| Directed (cyclic) | All four possible |

---

## 💡 Key Takeaways

- **Back edge = cycle** — most important application
- Undirected graphs only have Tree and Back edges
- Forward and Cross edges exist **only in directed** graphs
- Use discovery/finish times to classify edges
