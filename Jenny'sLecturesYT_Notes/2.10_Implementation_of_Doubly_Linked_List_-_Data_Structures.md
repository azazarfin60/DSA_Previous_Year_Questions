## 📌 Executive Summary

This lecture covers the **C implementation** of a Doubly Linked List — creating nodes, linking them with both prev and next pointers, and displaying the list in both forward and backward directions.

---

## 💻 Node Definition
```c
struct node {
    struct node *prev;
    int data;
    struct node *next;
};
struct node *head = NULL;
```

---

## 💻 Creating a DLL
```c
void createDLL(int n) {
    struct node *newNode, *temp;

    for (int i = 0; i < n; i++) {
        newNode = (struct node *)malloc(sizeof(struct node));
        printf("Enter data: ");
        scanf("%d", &newNode->data);
        newNode->next = NULL;

        if (head == NULL) {
            newNode->prev = NULL;
            head = newNode;
        } else {
            temp = head;
            while (temp->next != NULL)
                temp = temp->next;
            temp->next = newNode;
            newNode->prev = temp;
        }
    }
}
```

---

## 💻 Display Forward
```c
void displayForward() {
    struct node *temp = head;
    while (temp != NULL) {
        printf("%d ⇄ ", temp->data);
        temp = temp->next;
    }
    printf("NULL\n");
}
```

## 💻 Display Backward
```c
void displayBackward() {
    struct node *temp = head;
    // Go to last node
    while (temp->next != NULL)
        temp = temp->next;
    // Traverse backward
    while (temp != NULL) {
        printf("%d ⇄ ", temp->data);
        temp = temp->prev;
    }
    printf("NULL\n");
}
```

---

## 💡 Key Takeaways

- Each new node's `prev` and `next` must both be set correctly
- Forward display: traverse using `temp->next`
- Backward display: first go to end, then traverse using `temp->prev`
- `sizeof(struct node)` = 12 bytes (prev + data + next)
