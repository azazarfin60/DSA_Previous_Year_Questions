# 2.10 Implementation of Doubly Linked List (Creation & Display) | Data Structures

📺 [Watch on YouTube](https://www.youtube.com/watch?v=115K8LwP-xY) *(Note: Placeholder link)*

---

## 📌 Executive Summary
Implementing a Doubly Linked List (DLL) is very similar to implementing a Singly Linked List (SLL). The primary difference during the creation phase is that whenever we append a new node, we must establish **two links** instead of one: a forward link (`next`) from the previous node to the new node, and a backward link (`prev`) from the new node back to the previous node.

---

## 🛠️ The 3 Essential Components

### 1. Defining the Node Structure
A DLL node requires three fields:
```c
struct node {
    int data;               // Stores the actual value
    struct node *prev;      // Pointer to the previous node
    struct node *next;      // Pointer to the next node
};
```

### 2. Required Pointers
- `head`: Strictly points to the very first node.
- `newnode`: Holds the dynamically allocated memory for the node currently being created.
- `temp`: A multi-purpose pointer used to keep track of the *last* node during list creation, and to traverse the list during display.

### 3. Dynamic Memory Allocation
```c
newnode = (struct node*) malloc(sizeof(struct node));
```

---

## 💻 Full C Implementation: Create and Display

Below is the standard, exam-ready implementation for creating a doubly linked list dynamically.

```c
#include <stdio.h>
#include <stdlib.h>

// 1. Define the structure
struct node {
    int data;
    struct node *prev;
    struct node *next;
};

int main() {
    struct node *head = NULL, *newnode, *temp;
    int choice = 1;

    // ----------------------------------------
    // Phase 1: CREATION
    // ----------------------------------------
    while (choice) {
        // Allocate memory for the new node
        newnode = (struct node*) malloc(sizeof(struct node));
        
        printf("Enter data: ");
        scanf("%d", &newnode->data);
        
        // Initialize BOTH pointers to NULL
        newnode->prev = NULL;
        newnode->next = NULL; 

        // If it's the very first node
        if (head == NULL) {
            head = temp = newnode;
        } 
        // If the list already has nodes
        else {
            temp->next = newnode; // Establish the FORWARD link
            newnode->prev = temp; // Establish the BACKWARD link
            temp = newnode;       // Move temp to the new last node
        }

        printf("Do you want to continue? (0,1): ");
        scanf("%d", &choice);
    }

    // ----------------------------------------
    // Phase 2: DISPLAY (Forward Traversal)
    // ----------------------------------------
    // Note: Forward traversal code is EXACTLY the same as Singly Linked List
    temp = head; // Reset temp to the start of the list
    printf("\nDoubly Linked List Data: ");
    
    while (temp != NULL) {
        printf("%d <-> ", temp->data);
        temp = temp->next; // Move temp to the next node
    }
    printf("NULL\n");

    return 0;
}
```

---

## 🚨 Exam Focus: The Backward Link
If an exam asks to highlight the fundamental code difference between creating an SLL and a DLL, point directly to the `else` block:

```c
// Singly Linked List Append:
temp->next = newnode;
temp = newnode;

// Doubly Linked List Append:
temp->next = newnode;
newnode->prev = temp;   // <--- THIS IS THE ONLY DIFFERENCE
temp = newnode;
```
Forgetting `newnode->prev = temp;` degrades your Doubly Linked List into a bloated Singly Linked List, rendering backward traversal impossible.
