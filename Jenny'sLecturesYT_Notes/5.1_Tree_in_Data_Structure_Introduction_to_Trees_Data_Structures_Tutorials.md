## 🔑 Key Terminology
| Term | Definition |
|------|-----------|
| **Root** | Top node (no parent) |
| **Leaf** | Node with no children |
| **Parent** | Node directly above |
| **Child** | Node directly below |
| **Sibling** | Nodes sharing same parent |
| **Depth** | Distance from root (root = 0) |
| **Height** | Distance to deepest leaf |
| **Degree** | Number of children |
| **Level** | Depth + 1 (root = level 1) |
| **Subtree** | Tree formed by any node and its descendants |

## 📊 Tree vs Linear DS
| Feature | Linear (Array/LL) | Tree |
|---------|-------------------|------|
| Structure | Sequential | Hierarchical |
| Traversal | O(n) | O(log n) possible |
| Search | O(n) | O(log n) in BST |
| Examples | Array, LL, Stack | BST, AVL, Heap |
