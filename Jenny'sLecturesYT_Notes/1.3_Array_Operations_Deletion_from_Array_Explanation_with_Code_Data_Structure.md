## 📌 Executive Summary

This lecture covers **deletion** from an array at a specific position, at the beginning, and at the end. It explains the shifting process needed to maintain contiguous storage, provides complete C code, and discusses the O(1) optimization for unsorted arrays.

---

## 🔑 Deletion Process
1. Validate position (not out of bounds, array not empty)
2. **Shift elements left** from position to end
3. Decrement size

### Visual Example
```
Before: [6, 2, 10, 4, 5]   Delete from position 2 (index 1)
         0  1   2  3  4     size = 5

Step 1: Shift left starting from position
        a[1] = a[2]  →  [6, 10, 10, 4, 5]
        a[2] = a[3]  →  [6, 10,  4, 4, 5]
        a[3] = a[4]  →  [6, 10,  4, 5, 5]

Step 2: size-- → size = 4

Result: [6, 10, 4, 5]   (size = 4)
```

## 💻 Deletion at Specific Position

```c
int pos, item;

printf("Enter position to delete: ");
scanf("%d", &pos);

if (pos <= 0 || pos > size) {
    printf("Invalid position!");
} else if (size == 0) {
    printf("Underflow! Array is empty.");
} else {
    item = a[pos - 1];  // Store deleted value (optional)

    for (int i = pos - 1; i < size - 1; i++)
        a[i] = a[i + 1];    // Shift elements left
    size--;

    printf("Deleted value: %d\n", item);
}
```

---

## 💻 Deletion at Beginning

```c
item = a[0];
for (int i = 0; i < size - 1; i++)
    a[i] = a[i + 1];    // Shift ALL elements left
size--;
```

## 💻 Deletion at End

```c
item = a[size - 1];
size--;    // Simply reduce size — no shifting needed!
```

---

## ⏱️ Time Complexity

| Operation | Time Complexity |
|-----------|----------------|
| Delete from **end** | O(1) |
| Delete from **beginning** | O(n) |
| Delete from **position p** | O(n - p) |
| General worst case | **O(n)** |

### Unsorted Array Optimization — O(1)
```c
a[pos - 1] = a[size - 1];   // Replace with last element
size--;
// Time: O(1) — constant time!
```

> ⚠️ Only works for **unsorted** arrays. For **sorted** arrays, shifting is mandatory.

---

## 💡 Key Takeaways

- Deletion requires **left-shifting** elements to fill the gap
- Always validate: **position bounds** + **underflow** (empty array)
- Delete from end = O(1), delete from beginning = O(n)
- **Best algorithm for unsorted array: O(1)** — replace with last element
