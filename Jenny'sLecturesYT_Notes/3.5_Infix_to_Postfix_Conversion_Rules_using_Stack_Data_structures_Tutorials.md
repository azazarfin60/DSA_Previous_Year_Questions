## 📐 Algorithm: Infix → Postfix

1. Scan left to right
2. **Operand** → output directly
3. **`(`** → push to stack
4. **`)`** → pop and output until `(` found, discard `(`
5. **Operator** →
   - While stack top has **higher or equal precedence**: pop and output
   - Then push current operator
6. End → pop all remaining from stack to output

### Precedence: `^` > `*,/` > `+,-`
### Associativity: `^` is **Right-to-Left**, all others Left-to-Right

---

## 📐 Example: `A + B * C - D`

| Symbol | Stack | Output |
|--------|-------|--------|
| A | | A |
| + | + | A |
| B | + | A B |
| * | + * | A B |
| C | + * | A B C |
| - | - | A B C * + |
| D | - | A B C * + D |
| End | | A B C * + D - |

**Postfix: `A B C * + D -`** ✅
