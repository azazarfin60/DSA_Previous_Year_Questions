# 7.9 Heap Sort | Heapify Method | Build Max Heap Algorithm | Sorting Algorithms

📺 [Watch on YouTube](https://www.youtube.com/watch?v=Q_eia3jC9Ts)

## 📐 Algorithm
1. **Build Max Heap** from array (heapify from last non-leaf to root)
2. Repeat: swap root (max) with last → reduce heap size → heapify root

## 💻 C Code
```c
void heapify(int arr[], int n, int i) {
    int largest = i, left = 2*i+1, right = 2*i+2;
    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;
    if (largest != i) {
        int temp = arr[i]; arr[i] = arr[largest]; arr[largest] = temp;
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    // Build max heap
    for (int i = n/2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    // Extract elements one by one
    for (int i = n-1; i > 0; i--) {
        int temp = arr[0]; arr[0] = arr[i]; arr[i] = temp;
        heapify(arr, i, 0);
    }
}
```

## 📐 Trace: [4, 10, 3, 5, 1]
```
Build Heap: [10, 5, 3, 4, 1]
Swap 10↔1:  [1, 5, 3, 4, 10] → heapify → [5, 4, 3, 1, 10]
Swap 5↔1:   [1, 4, 3, 5, 10] → heapify → [4, 1, 3, 5, 10]
Swap 4↔3:   [3, 1, 4, 5, 10] → heapify → [3, 1, 4, 5, 10]
Swap 3↔1:   [1, 3, 4, 5, 10] ✅
```
