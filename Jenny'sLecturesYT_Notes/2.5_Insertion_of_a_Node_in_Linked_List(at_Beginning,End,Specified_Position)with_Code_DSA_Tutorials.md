## 📌 Executive Summary

This lecture covers **three types of insertion** in a singly linked list: at the beginning, at the end, and after a given position. Each case includes detailed pointer manipulation and the **critical rule** of updating the right link first.

---

## 💻 1. Insertion at Beginning

### Steps
1. Create a new node using `malloc`
2. **Point new node's next to current head** (right link first!)
3. **Update head to point to new node**

```c
newNode->next = head;   // Step 1: new node → old first
head = newNode;         // Step 2: head → new node
```

### Visual
```
Before:  Head → [7|200] → [1|300] → [9|NULL]
After:   Head → [2|100] → [7|200] → [1|300] → [9|NULL]
```

**Time: O(1)**

---

## 💻 2. Insertion at End

### Steps
1. Create new node, set `next = NULL`
2. Traverse to **last node** using temp (while `temp->next != NULL`)
3. Set last node's `next` to new node

```c
newNode->next = NULL;
temp = head;
while (temp->next != NULL)    // NOT temp != NULL
    temp = temp->next;
temp->next = newNode;
```

> ⚠️ Use `temp->next != NULL` (not `temp != NULL`). If temp becomes NULL, can't access `temp->next`.

**Time: O(n)** — must traverse to last node

---

## 💻 3. Insertion After a Given Position

### Steps
1. Traverse to the specified position using counter
2. **Right link first:** `newNode->next = temp->next`
3. **Left link second:** `temp->next = newNode`

```c
temp = head;
for (int i = 1; i < pos; i++)
    temp = temp->next;

newNode->next = temp->next;  // Step 1: right link
temp->next = newNode;        // Step 2: left link
```

**Time: O(n)**

---

## ⚠️ Critical Rule: Order of Pointer Updates

```
✅ CORRECT:  newNode->next = temp->next  THEN  temp->next = newNode
❌ WRONG:    temp->next = newNode  THEN  newNode->next = temp->next
              (loses reference to rest of list!)
```

---

## ⏱️ Time Complexity

| Operation | Time |
|-----------|------|
| Insert at **beginning** | **O(1)** |
| Insert at **end** | **O(n)** |
| Insert after **position p** | **O(p)** |

---

## 💡 Key Takeaways

- Always **create node** first, then insert
- **Right link first, left link second** — critical!
- Use `temp->next != NULL` for end insertion
- Validate position before insertion
- Declare `struct node` and pointers **globally** for multi-function use
