## 📌 Executive Summary

This lecture covers the **Doubly Circular Linked List (DCLL)** — a list where each node has prev and next pointers, and the list forms a **circle in both directions**. Key advantage: `head->prev` gives **O(1)** access to last node.

---

## 🔑 Structure
```
[300|1|200] ⇄ [100|2|300] ⇄ [200|3|100]
First's prev = last, Last's next = first
```

### C Definition
```c
struct node {
    struct node *prev;
    int data;
    struct node *next;
};
struct node *head = NULL;
```

---

## 💻 Creation
```c
void create(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;

    if (head == NULL) {
        head = newNode;
        newNode->next = head;
        newNode->prev = head;   // Points to itself
    } else {
        struct node *last = head->prev;  // O(1) last access!
        newNode->next = head;
        newNode->prev = last;
        last->next = newNode;
        head->prev = newNode;
    }
}
```

## 💻 Display
```c
void display() {
    if (head == NULL) return;
    struct node *temp = head;
    do {
        printf("%d ⇄ ", temp->data);
        temp = temp->next;
    } while (temp != head);
}
```

---

## 🎯 Key Advantage
> `head->prev` = last node → **O(1)** access to both ends!

---

## 💡 Key Takeaways

- DCLL = DLL + Circular (best of both worlds)
- Single-node: `next = prev = head` (points to itself both ways)
- `head->prev` = last node — no traversal needed
- All boundary operations possible in **O(1)**
