## 📌 Executive Summary
A **Doubly Circular Linked List** combines the bidirectional traversal capabilities of a Doubly Linked List with the closed-loop structure of a Circular Linked List. 

The defining characteristic of this list is the **total absence of `NULL` pointers**.
- The `next` pointer of the last node points to the first node.
- The `prev` pointer of the first node points to the last node.

---

## 🛠️ Phase 1: Creation

When creating a Doubly Circular Linked List and appending nodes to the end, we must update **five pointers** to maintain both the birectional links and the circular closed loop.

*Note: For clarity, this implementation maintains both `head` and `tail` pointers.*

### The 5 Crucial Link Updates (When list is not empty):
1. **Connect old tail forward:** `tail->next = newnode;`
2. **Connect newnode backward:** `newnode->prev = tail;`
3. **Secure circular forward link:** `newnode->next = head;`
4. **Secure circular backward link:** `head->prev = newnode;`
5. **Shift global tail:** `tail = newnode;`

### C Code Snippet
```c
struct node {
    int data;
    struct node *prev;
    struct node *next;
};
struct node *head = NULL, *tail = NULL;

void createDCLL() {
    struct node *newnode;
    int choice = 1;

    while (choice) {
        newnode = (struct node*) malloc(sizeof(struct node));
        printf("Enter data: ");
        scanf("%d", &newnode->data);

        // 1. If list is empty (Create circle of 1)
        if (head == NULL) {
            head = tail = newnode;
            head->next = head; // Points to itself
            head->prev = head; // Points to itself
        } 
        // 2. If list already has nodes (The 5 pointer updates)
        else {
            tail->next = newnode;
            newnode->prev = tail;
            newnode->next = head;
            head->prev = newnode;
            tail = newnode;
        }
        
        printf("Continue? (0,1): ");
        scanf("%d", &choice);
    }
}
```

> [!TIP]
> **Exam Validation Trick:**
> If you want to prove to the examiner that your list is truly a doubly circular linked list, you can show how to access the first node using the tail pointer, and the last node using the head pointer:
> - `tail->next->data` will print the first element.
> - `head->prev->data` will print the last element.

## 🛠️ Phase 2: Display / Traversal

Because there are no `NULL` pointers, a standard `while(temp != NULL)` loop will result in an infinite loop. We must traverse until we reach the end of the circle.

Just as we did with the Singly Circular Linked List, utilizing a **`do-while` loop** is the most elegant solution. It executes the print statement for the `head` node immediately, and gracefully terminates exactly after processing the `tail` node, avoiding any "off-by-one" output errors.

### C Code Snippet

```c
void displayDCLL() {
    struct node *temp;
    
    if (head == NULL) {
        printf("List is empty.\n");
        return;
    }
    
    temp = head;
    
    // ----------------------------------------------------
    // Using do-while prevents the off-by-one printing trap
    // ----------------------------------------------------
    do {
        printf("%d <-> ", temp->data);
        temp = temp->next;
    } while (temp != head); // Stop when we circle back to the start
    
    printf("...back to head\n");
}
```

> [!CAUTION]
> **Exam Trap (Standard While Loop Fallback):**
> If you choose to use a standard `while` loop, your condition will be `while (temp != tail)`. However, doing so means the loop will exit *before* printing the data in the `tail` node. If you do this, you **must** remember to write an extra `printf("%d", temp->data);` statement immediately after the loop closes to avoid losing marks for incomplete output.
