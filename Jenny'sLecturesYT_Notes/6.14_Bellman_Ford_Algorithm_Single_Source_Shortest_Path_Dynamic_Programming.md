# 6.14 Bellman Ford Algorithm - Single Source Shortest Path | Dynamic Programming

📺 [Watch on YouTube](https://www.youtube.com/watch?v=KudAWAMiQog)

---

## 📐 Algorithm
1. Set `dist[source] = 0`, all others `= ∞`
2. Repeat **V-1 times**: for every edge (u,v,w), if `dist[u] + w < dist[v]` → update `dist[v]`
3. **One more pass** to detect negative cycles: if any dist reduces → negative cycle exists!

## 📐 Example
```
Graph: 0→1(6), 0→2(7), 1→2(8), 1→3(5), 1→4(-4),
       2→3(-3), 2→4(9), 3→1(-2), 4→3(7)
Source = 0

Initial: [0, ∞, ∞, ∞, ∞]

After V-1 = 4 iterations:
dist = [0, 2, 7, 4, -2]
```

| Step | Edge Relaxed | dist[] |
|------|-------------|--------|
| Init | — | [0, ∞, ∞, ∞, ∞] |
| Pass 1 | 0→1, 0→2 | [0, 6, 7, ∞, ∞] |
| Pass 2 | 1→3, 1→4, 2→3 | [0, 6, 7, 4, 2] |
| ... | Continue | [0, 2, 7, 4, -2] |

## 📊 Bellman-Ford vs Dijkstra
| Feature | Dijkstra | Bellman-Ford |
|---------|----------|-------------|
| Negative edges | ❌ | ✅ |
| Negative cycle detect | ❌ | ✅ |
| Time | O(V²) or O(E log V) | **O(VE)** |
| Approach | Greedy | DP |

## ⏱️ Time: **O(V × E)**, Space: O(V)
