## 💻 Complete C Implementation
```c
#define MAX 100
int stack[MAX], top = -1;

void push(int data) {
    if (top == MAX - 1) { printf("Overflow!\n"); return; }
    stack[++top] = data;
}

int pop() {
    if (top == -1) { printf("Underflow!\n"); return -1; }
    return stack[top--];
}

int peek() {
    if (top == -1) return -1;
    return stack[top];
}

int isEmpty() { return top == -1; }
int isFull() { return top == MAX - 1; }

void display() {
    for (int i = top; i >= 0; i--)
        printf("%d\n", stack[i]);
}
```

## 💡 Key: `top = -1` means empty. All operations O(1).
