# 5.17 Red Black Tree Insertion | Insertion Algorithm | Data Structure Tutorials

📺 [Watch on YouTube](https://www.youtube.com/watch?v=qA02XWRTBdw)

## 🔑 RBT Properties
1. Every node is **Red or Black**
2. Root is always **Black**
3. No two adjacent **Red** nodes (Red parent cannot have Red child)
4. Every path from root to NULL has **same number of Black nodes**
5. NULL nodes are **Black**

## 📐 Insertion Algorithm
1. Insert as in BST, color new node **Red**
2. If parent is **Black** → done
3. If parent is **Red** → Fix:
   - **Case 1**: Uncle is Red → recolor parent, uncle to Black, grandparent to Red, move up
   - **Case 2**: Uncle is Black, node is inner child → rotate to make outer child (Case 3)
   - **Case 3**: Uncle is Black, node is outer child → rotate grandparent, swap colors of parent & grandparent
4. Ensure root is **Black**

## 📐 Example: Insert 10, 18, 7, 15, 16

```
Insert 10:    10(B)        ← root is always Black

Insert 18:    10(B)
                \
                18(R)      ← parent is Black, done

Insert 7:     10(B)
             /    \
           7(R)  18(R)     ← parent is Black, done

Insert 15:    Parent=18(R), Uncle=7(R) → Case 1: Recolor
              10(B)                    10(B)
             /    \        →          /    \
           7(R)  18(R)              7(B)  18(B)
                 /                        /
               15(R)                    15(R)

Insert 16:    Parent=15(R), Uncle=NULL(B) → Case 2 (LR at 18)
              First: Left rotate on 15
              10(B)                    10(B)
             /    \        →          /    \
           7(B)  18(B)              7(B)  18(B)
                 /                        /
               15(R)                    16(R)
                 \                      /
                 16(R)                15(R)

              Then: Case 3 → Right rotate on 18, recolor
              10(B)
             /    \
           7(B)  16(B)
                /    \
              15(R)  18(R)
```

## ⏱️ All operations: **O(log n)**. Max 2 rotations per insert.
