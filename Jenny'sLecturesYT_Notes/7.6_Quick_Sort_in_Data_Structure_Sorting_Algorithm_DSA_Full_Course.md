## 📐 Algorithm
1. Choose **pivot** (typically last element)
2. **Partition**: place pivot at correct position, all smaller to left, larger to right
3. Recursively sort left and right partitions

## 💻 C Code
```c
int partition(int arr[], int low, int high) {
    int pivot = arr[high];   // Last element as pivot
    int i = low - 1;

    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
        }
    }
    int temp = arr[i+1]; arr[i+1] = arr[high]; arr[high] = temp;
    return i + 1;
}

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```


## ⏱️ Complexity
| Case | Time |
|------|------|
| **Best** | O(n log n) |
| **Average** | O(n log n) |
| **Worst** | **O(n²)** (already sorted, poor pivot) |

**Space: O(log n)** recursion stack | **Stable: ❌ No** | **In-place: ✅ Yes**

## 💡 Fastest in practice. Worst case avoided with random pivot selection.
