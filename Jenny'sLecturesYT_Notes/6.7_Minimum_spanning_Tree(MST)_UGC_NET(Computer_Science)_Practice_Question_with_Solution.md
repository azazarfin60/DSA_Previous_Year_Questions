## 📌 Executive Summary

This lecture solves MST **practice problems** — finding the number of possible MSTs, applying Prim's and Kruskal's to exam-style graphs, and handling edge weight ties.

---

## 🔑 Key Exam Concepts

### When Multiple MSTs Exist
> If two or more edges have the **same weight** and can be interchanged, multiple MSTs are possible with the **same total weight**.

### Uniqueness Theorem
> If all edge weights are **distinct**, the MST is **unique**.

## 📐 Practice: Find MST Weight

```
Graph:
  A --1-- B --4-- C
  |       |       |
  3       2       5
  |       |       |
  D --6-- E --7-- F
```

### Using Kruskal's (sort edges, pick smallest, avoid cycles):

| Step | Edge | Weight | Include? |
|------|------|--------|----------|
| 1 | A-B | 1 | ✅ |
| 2 | B-E | 2 | ✅ |
| 3 | A-D | 3 | ✅ |
| 4 | B-C | 4 | ✅ |
| 5 | C-F | 5 | ✅ |
| 6 | D-E | 6 | ❌ (cycle) |
| 7 | E-F | 7 | ❌ (cycle) |

**MST weight = 1 + 2 + 3 + 4 + 5 = 15**
**Edges in MST = 5 (= V-1 = 6-1)**

---

## 📐 How to Count Number of MSTs

1. Apply Kruskal's normally
2. When you encounter edges with **same weight** that are interchangeable → each choice gives a different MST
3. Multiply the choices

---

## 💡 Key Takeaways

- MST is unique when all weights are distinct
- Same weight edges → multiple MSTs possible (same total weight)
- Always verify: V-1 edges, no cycles, minimum total weight
- Practice both Prim's and Kruskal's — exam may specify one
