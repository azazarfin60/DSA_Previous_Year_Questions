## 📐 Prefix to Infix (Scan Right to Left)
1. Scan **right to left**
2. Operand → push as string
3. Operator → pop two, combine as `(op1 ○ op2)`, push result

### Example: `* + A B - C D`
| R→L | Stack |
|-----|-------|
| D | ["D"] |
| C | ["D","C"] |
| - | ["(C-D)"] |
| B | ["(C-D)","B"] |
| A | ["(C-D)","B","A"] |
| + | ["(C-D)","(A+B)"] |
| * | ["((A+B)*(C-D))"] |

**Infix: `(A+B)*(C-D)`** ✅

---

## 📐 Postfix to Infix (Scan Left to Right)
1. Scan **left to right**
2. Operand → push
3. Operator → pop two, combine as `(op1 ○ op2)`, push

### Example: `A B + C D - *`
| L→R | Stack |
|-----|-------|
| A | ["A"] |
| B | ["A","B"] |
| + | ["(A+B)"] |
| C | ["(A+B)","C"] |
| D | ["(A+B)","C","D"] |
| - | ["(A+B)","(C-D)"] |
| * | ["((A+B)*(C-D))"] |

**Infix: `(A+B)*(C-D)`** ✅
