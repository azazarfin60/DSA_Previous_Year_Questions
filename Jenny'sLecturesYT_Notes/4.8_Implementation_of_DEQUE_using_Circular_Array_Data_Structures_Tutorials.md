# 4.8 DEQUE Implementation using Circular Arrays | Data Structures

📺 [Watch on YouTube](https://www.youtube.com/watch?v=115K8LwP-xY) *(Note: Placeholder link)*

---

## 📌 Executive Summary
Implementing a Double Ended Queue (DEQUE) using a circular array allows us to achieve $O(1)$ time complexity for insertions and deletions at **both** ends, without wasting any memory space. 

Because we are moving pointers both forward (`++`) and backward (`--`), we must handle specific array boundary wrap-around conditions.

---

## 🛠️ Global Setup and Core Conditions

```c
#include <stdio.h>
#define N 5

int deque[N];
int front = -1, rear = -1;
```

### The "Queue is Full" Condition (Overflow)
A DEQUE in a circular array is full under two conditions:
1. `front == 0 && rear == N - 1` (Linear full state)
2. `front == rear + 1` (Circular full state: the rear pointer has wrapped around and caught up to just behind the front pointer)

### The "Queue is Empty" Condition (Underflow)
`front == -1 && rear == -1`

---

## 1️⃣ Enqueue Operations (Insertion)

### A. Insert at Front (`enqueueFront`)
Normally, we insert at the rear. When inserting at the front, the `front` pointer steps **backward**.
**Wrap-Around Rule:** If `front` is 0, stepping backward means it must wrap around to `N - 1`.

```c
void enqueueFront(int x) {
    if ((front == 0 && rear == N - 1) || (front == rear + 1)) {
        printf("Overflow: DEQUE is full\n");
    } 
    else if (front == -1 && rear == -1) { // Empty queue
        front = rear = 0;
        deque[front] = x;
    } 
    else if (front == 0) { // Wrap backward
        front = N - 1;
        deque[front] = x;
    } 
    else { // Normal backward step
        front--;
        deque[front] = x;
    }
}
```

### B. Insert at Rear (`enqueueRear`)
This is identical to a standard Circular Queue enqueue operation. The `rear` pointer steps **forward**.
**Wrap-Around Rule:** If `rear` is `N - 1`, stepping forward means it wraps to 0.

```c
void enqueueRear(int x) {
    if ((front == 0 && rear == N - 1) || (front == rear + 1)) {
        printf("Overflow: DEQUE is full\n");
    } 
    else if (front == -1 && rear == -1) { // Empty queue
        front = rear = 0;
        deque[rear] = x;
    } 
    else if (rear == N - 1) { // Wrap forward
        rear = 0;
        deque[rear] = x;
    } 
    else { // Normal forward step
        rear++;
        deque[rear] = x;
    }
}
```

---

## 2️⃣ Dequeue Operations (Deletion)

### A. Delete from Front (`dequeueFront`)
This is identical to a standard Circular Queue dequeue operation. The `front` pointer steps **forward**.
**Wrap-Around Rule:** If `front` is `N - 1`, stepping forward means it wraps to 0.

```c
void dequeueFront() {
    if (front == -1 && rear == -1) {
        printf("Underflow: DEQUE is empty\n");
    } 
    else if (front == rear) { // Only one element left
        printf("Dequeued: %d\n", deque[front]);
        front = rear = -1;
    } 
    else if (front == N - 1) { // Wrap forward
        printf("Dequeued: %d\n", deque[front]);
        front = 0;
    } 
    else { // Normal forward step
        printf("Dequeued: %d\n", deque[front]);
        front++;
    }
}
```

### B. Delete from Rear (`dequeueRear`)
Normally, we delete from the front. When deleting from the rear, the `rear` pointer steps **backward**.
**Wrap-Around Rule:** If `rear` is 0, stepping backward means it must wrap around to `N - 1`.

```c
void dequeueRear() {
    if (front == -1 && rear == -1) {
        printf("Underflow: DEQUE is empty\n");
    } 
    else if (front == rear) { // Only one element left
        printf("Dequeued: %d\n", deque[rear]);
        front = rear = -1;
    } 
    else if (rear == 0) { // Wrap backward
        printf("Dequeued: %d\n", deque[rear]);
        rear = N - 1;
    } 
    else { // Normal backward step
        printf("Dequeued: %d\n", deque[rear]);
        rear--;
    }
}
```

## 3️⃣ Display Operation ($O(n)$)

The display function is completely identical to the standard Circular Queue display logic. We must use a `while` loop to traverse circularly from `front` to `rear`.

> [!CAUTION]
> **Exam Trap:** Don't forget to print the last element sitting at the `rear` index outside of the `while` loop!

```c
void display() {
    int i = front;
    
    if (front == -1 && rear == -1) {
        printf("DEQUE is empty\n");
        return;
    }
    
    printf("DEQUE elements: ");
    
    while (i != rear) {
        printf("%d ", deque[i]);
        i = (i + 1) % N; // Circular forward step
    }
    
    // Print the final element
    printf("%d\n", deque[rear]); 
}
```
