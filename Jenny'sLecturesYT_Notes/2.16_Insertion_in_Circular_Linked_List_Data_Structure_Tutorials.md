## 📌 Executive Summary

This lecture covers insertion in a CLL at the **beginning**, **end**, and **after a given position**. Key difference from SLL: must update the **last node's next** to maintain circular structure.

---

## 💻 1. Insert at Beginning
```c
void insertAtBeginning(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;

    if (head == NULL) {
        head = newNode;
        newNode->next = head;
        return;
    }
    // Find last node
    struct node *temp = head;
    while (temp->next != head)
        temp = temp->next;

    newNode->next = head;    // New → old head
    temp->next = newNode;    // Last → new node
    head = newNode;          // Update head
}
```
**Time: O(n)** — must find last node to update its link

---

## 💻 2. Insert at End
```c
void insertAtEnd(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;

    if (head == NULL) {
        head = newNode;
        newNode->next = head;
        return;
    }
    struct node *temp = head;
    while (temp->next != head)
        temp = temp->next;

    temp->next = newNode;     // Old last → new
    newNode->next = head;     // New last → head (circular)
}
```
**Time: O(n)**

---

## 💻 3. Insert After Position
```c
void insertAfterPos(int data, int pos) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;

    struct node *temp = head;
    for (int i = 1; i < pos; i++)
        temp = temp->next;

    newNode->next = temp->next;
    temp->next = newNode;
}
```
**Time: O(n)**

---

## 💡 Key Takeaways

- **Beginning insertion** in CLL is O(n) — must find and update last node
- This is a disadvantage compared to SLL (O(1) at beginning)
- Can optimize to O(1) by maintaining a **tail pointer**
