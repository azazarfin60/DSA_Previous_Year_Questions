# 8.2 Hashing - Quadratic Probing | Data Structures

📺 [Watch on YouTube](https://www.youtube.com/watch?v=115K8LwP-xY) *(Note: Placeholder link)*

---

## 📌 Executive Summary
**Quadratic Probing** is a closed hashing (open addressing) collision resolution technique. 

It was developed to solve the major drawback of Linear Probing known as **Primary Clustering**. In linear probing, if a cluster of contiguous array slots becomes filled, any new collision in that area will force the algorithm to step through *every single filled slot* one-by-one, severely degrading the expected $O(1)$ performance.

Quadratic probing solves this by using a quadratic polynomial ($i^2$) to calculate the next probe location, effectively causing the search to "jump" further and further away from the cluster to quickly find a free slot.

---

## 📐 The Quadratic Probing Formula

When a collision occurs at the ideal hash index $H(k)$, the next location to check is determined by:

**$H_i(k) = (H(k) + i^2) \pmod{M}$**

*Where:*
- $H(k)$ = The original hash index (the value of $u$).
- $i$ = The probe attempt number ($0, 1, 2, ..., M-1$).
- $M$ = The size of the hash table.

## 📝 Exam Walkthrough: Quadratic Probing

> **Problem Statement:**
> Store the keys `[3, 2, 9, 6, 11, 13, 7, 12]` into a Hash Table of size $M = 10$.
> Use the Hash Function: $H(k) = (2k + 3) \pmod{10}$.
> Resolve collisions using **Quadratic Probing**.

### Step 1: Initial Insertions (No Collisions)
Based on the hash function $H(k)$:
- $k=3 \rightarrow \text{Index } 9$ (Free)
- $k=2 \rightarrow \text{Index } 7$ (Free)
- $k=9 \rightarrow \text{Index } 1$ (Free)
- $k=6 \rightarrow \text{Index } 5$ (Free)

### Step 2: Resolving Collisions (The Quadratic Jumps)

**Inserting 11:** Ideal index $= 5$ (Collision with 6)
- $i=0$: $(5 + 0^2) \pmod{10} = 5$ (Full)
- $i=1$: $(5 + 1^2) \pmod{10} = \mathbf{6}$ (Free!). Insert 11 at **Index 6**.

**Inserting 13:** Ideal index $= 9$ (Collision with 3)
- $i=0$: $(9 + 0^2) \pmod{10} = 9$ (Full)
- $i=1$: $(9 + 1^2) \pmod{10} = 10 \pmod{10} = \mathbf{0}$ (Free!). Insert 13 at **Index 0**.

**Inserting 7:** Ideal index $= 7$ (Collision with 2)
- $i=0$: $(7 + 0^2) \pmod{10} = 7$ (Full)
- $i=1$: $(7 + 1^2) \pmod{10} = \mathbf{8}$ (Free!). Insert 7 at **Index 8**.

---

### 🔥 The Power of Quadratic Jumps (Inserting 12)

**Inserting 12:** Ideal index $= 7$ (Collision with 2 & 7).
Let's see how the quadratic formula bypasses clusters:

1. **Attempt 0 ($i=0$):** $(7 + 0) \pmod{10} = 7$ -> Full (Contains 2)
2. **Attempt 1 ($i=1$):** $(7 + 1) \pmod{10} = 8$ -> Full (Contains 7)
3. **Attempt 2 ($i=2$):** $(7 + 4) \pmod{10} = 1$ -> Full (Contains 9). 
   *Notice it completely skipped indices 9 and 0!*
4. **Attempt 3 ($i=3$):** $(7 + 9) \pmod{10} = 6$ -> Full (Contains 11). 
   *Notice it completely skipped indices 2, 3, 4, and 5!*
5. **Attempt 4 ($i=4$):** $(7 + 16) \pmod{10} = 23 \pmod{10} = \mathbf{3}$ -> **FREE!** Insert 12 at **Index 3**.

*Total Probes for 12:* **5 probes**. *(In linear probing, this exact same insertion took 6 probes).*
