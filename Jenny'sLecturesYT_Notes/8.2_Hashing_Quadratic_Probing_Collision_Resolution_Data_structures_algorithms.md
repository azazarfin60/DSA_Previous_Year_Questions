## 📐 Formula
```
h(key, i) = (h(key) + i²) % table_size     where i = 0, 1, 2, ...
```

## 📐 Example: Insert 37, 90, 55, 22, 17 into table size 10

```
h(37)=7, i=0: slot 7 → place ✅
h(90)=0, i=0: slot 0 → place ✅
h(55)=5, i=0: slot 5 → place ✅
h(22)=2, i=0: slot 2 → place ✅
h(17)=7, COLLISION!
  i=1: (7 + 1) % 10 = 8 → place ✅

Table: [90, _, 22, _, _, 55, _, 37, 17, _]
```

## 📊 Linear vs Quadratic Probing
| Feature | Linear | Quadratic |
|---------|--------|-----------|
| Probe | +1, +2, +3... | +1, +4, +9... |
| Clustering | **Primary** ⚠️ | **Secondary** (milder) |
| All slots visited? | ✅ Always | ❌ Not guaranteed |

## ⚠️ Quadratic probing may **fail to find empty slot** even if table isn't full. Use table size = **prime** and load factor < 0.5.
