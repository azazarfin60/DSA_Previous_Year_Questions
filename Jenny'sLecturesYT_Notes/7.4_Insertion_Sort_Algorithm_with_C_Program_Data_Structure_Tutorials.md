## 💻 C Code
```c
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];  // Shift right
            j--;
        }
        arr[j + 1] = key;         // Insert
    }
}
```

## 📐 Trace: [5, 3, 4, 1, 2]
```
i=1: key=3, shift 5 → [3, 5, 4, 1, 2]
i=2: key=4, shift 5 → [3, 4, 5, 1, 2]
i=3: key=1, shift 5,4,3 → [1, 3, 4, 5, 2]
i=4: key=2, shift 5,4,3 → [1, 2, 3, 4, 5] ✅
```

## ⏱️ Best: O(n) already sorted | Worst: O(n²) reverse | Stable: ✅ | In-place: ✅
