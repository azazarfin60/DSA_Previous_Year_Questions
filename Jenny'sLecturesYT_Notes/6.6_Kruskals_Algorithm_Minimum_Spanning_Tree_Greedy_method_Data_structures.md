# 6.6 Kruskal's Algorithm for Minimum Spanning Tree - Greedy Method | Data Structures

📺 [Watch on YouTube](https://www.youtube.com/watch?v=EjVHtpWkIho)

---

## 📐 Algorithm
1. **Sort** all edges by weight (ascending)
2. Pick smallest edge. If it doesn't form a **cycle** → add to MST
3. Use **Union-Find** (Disjoint Set) to detect cycles
4. Repeat until V-1 edges selected

## 📐 Example
```
Edges sorted: (A-B,1), (C-D,2), (B-C,3), (A-D,4), (B-D,5)

Step 1: A-B(1) → no cycle → add ✅
Step 2: C-D(2) → no cycle → add ✅
Step 3: B-C(3) → no cycle → add ✅ (MST complete: 3 edges for 4 vertices)
Step 4: A-D(4) → forms cycle → skip ❌

MST weight = 1 + 2 + 3 = 6
```

## 📐 Union-Find for Cycle Detection
```c
int parent[MAX];
int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]); // Path compression
    return parent[x];
}
void unionSet(int x, int y) {
    parent[find(x)] = find(y);
}
// Cycle check: if find(u) == find(v) → adding (u,v) creates cycle
```

## ⏱️ Time: **O(E log E)** (dominated by sorting)
