# 2.17 Deletion of a Node from Circular Linked List (beginning, end, given position) | DSA Tutorials

## 💻 1. Delete from Beginning
```c
void deleteFromBeginning() {
    if (head == NULL) return;
    if (head->next == head) { free(head); head = NULL; return; }
    struct node *temp = head, *last = head;
    while (last->next != head) last = last->next;
    head = head->next;
    last->next = head;
    free(temp);
}
```

## 💻 2. Delete from End
```c
void deleteFromEnd() {
    if (head == NULL) return;
    if (head->next == head) { free(head); head = NULL; return; }
    struct node *temp = head, *prev;
    while (temp->next != head) { prev = temp; temp = temp->next; }
    prev->next = head;
    free(temp);
}
```

## 💻 3. Delete from Position
```c
void deleteFromPos(int pos) {
    if (pos == 1) { deleteFromBeginning(); return; }
    struct node *temp = head;
    for (int i = 1; i < pos - 1; i++) temp = temp->next;
    struct node *del = temp->next;
    temp->next = del->next;
    free(del);
}
```

**Key: Must update last node's `next` to maintain circular link.**
