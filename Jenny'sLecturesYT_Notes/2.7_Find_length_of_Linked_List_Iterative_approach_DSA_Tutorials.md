## 💻 Iterative Length Calculation
```c
int length(struct node *head) {
    int count = 0;
    struct node *temp = head;
    while (temp != NULL) {
        count++;
        temp = temp->next;
    }
    return count;
}
```

## 💻 Recursive Version
```c
int lengthRec(struct node *head) {
    if (head == NULL) return 0;
    return 1 + lengthRec(head->next);
}
```

**Time: O(n), Space: O(1) iterative / O(n) recursive**
