# 5.5 Binary Tree Traversals (Inorder, Preorder and Postorder) | Data structures and algorithms

📺 [Watch on YouTube](https://www.youtube.com/watch?v=-b2lciNd2L4)

---

## 📐 Three DFS Traversals

| Traversal | Order | Mnemonic |
|-----------|-------|----------|
| **Inorder** | Left → Root → Right | LNR |
| **Preorder** | Root → Left → Right | NLR |
| **Postorder** | Left → Right → Root | LRN |

## 📐 Example
```
       1
      / \
     2   3
    / \
   4   5

Inorder:   4, 2, 5, 1, 3
Preorder:  1, 2, 4, 5, 3
Postorder: 4, 5, 2, 3, 1
```

## 💻 C Code
```c
void inorder(struct node *root) {
    if (root == NULL) return;
    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}

void preorder(struct node *root) {
    if (root == NULL) return;
    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}

void postorder(struct node *root) {
    if (root == NULL) return;
    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->data);
}
```

## 💡 Inorder of BST gives **sorted order**. Time: O(n) for all.
