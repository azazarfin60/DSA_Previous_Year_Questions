## 🔑 MST Properties
1. A spanning tree of V vertices has exactly **V-1 edges**
2. MST weight is **minimum** among all spanning trees
3. If all edges have distinct weights → MST is **unique**
4. **Cut property**: Minimum weight edge crossing any cut must be in MST
5. **Cycle property**: Maximum weight edge in any cycle is NOT in MST
6. A connected graph with V vertices and V-1 edges is a tree

## 📊 Algorithms
| Algorithm | Approach | Best for | Time |
|-----------|----------|----------|------|
| **Prim's** | Grow from vertex | Dense graphs | O(V²) |
| **Kruskal's** | Sort edges, add smallest | Sparse graphs | O(E log E) |
