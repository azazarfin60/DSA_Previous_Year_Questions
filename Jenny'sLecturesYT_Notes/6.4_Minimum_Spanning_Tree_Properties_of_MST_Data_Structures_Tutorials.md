# 6.4 Minimum Spanning Tree & Its Properties | Data Structures Tutorials

📺 [Watch on YouTube](https://www.youtube.com/watch?v=RjcjXnS51eQ)

---

## 📌 Executive Summary
A **Spanning Tree (ST)** of a connected, undirected graph is a subgraph that includes all the vertices of the original graph but has the minimum possible number of edges to keep those vertices connected. 

If the graph is weighted, a **Minimum Spanning Tree (MST)** is the specific spanning tree whose total sum of edge weights is the absolute minimum possible compared to all other spanning trees.

---

## 🔑 Core Properties of Spanning Trees
For any connected graph $G$ with $V$ vertices and $E$ edges, its Spanning Tree $T$ with $V'$ vertices and $E'$ edges mathematically satisfies:
1. **Vertices Constraint:** $V' = V$ (It contains exactly the same vertices as the original graph).
2. **Edges Constraint:** $E' = V - 1$ (It contains exactly $V-1$ edges).
3. **No Cycles:** A spanning tree must not contain any loops/cycles.
4. **Connectedness:** A spanning tree must not be disconnected.

### ⚠️ Critical Edge Manipulation Rules
- **Adding an Edge:** If you add even a single edge to a valid spanning tree, it will immediately **create a cycle (loop)**.
- **Removing an Edge:** If you remove even a single edge from a valid spanning tree, the graph will immediately become **disconnected**.

---

## 🌲 Properties of the Minimum Spanning Tree (MST)
1. **Definition:** Out of all possible spanning trees for a weighted graph, the MST is the one where the sum of its edge weights is minimized.
2. **Uniqueness Property:** If every edge in the graph has a **distinct (unique) weight**, there will be exactly **one unique** Minimum Spanning Tree. 
   - *Note: If some edge weights are identical, the graph can have multiple different MSTs, but their total minimum cost will still be exactly the same.*

---

## 🧮 Mathematical & Graph-Specific Properties
1. **Cayley's Formula:** A Complete Undirected Graph with $N$ vertices can have a maximum of **$N^{N-2}$** different spanning trees. 
   - *Example: A complete graph with 4 vertices has $4^{4-2} = 4^2 = 16$ possible spanning trees.*
2. **Edge Removal Formula:** From a complete graph with $E$ edges and $N$ vertices, you must remove exactly **$E - N + 1$** edges to construct a valid spanning tree.
3. **Connectivity Rule:** Every connected, undirected graph is guaranteed to have at least one spanning tree.
4. **Disconnected Graphs:** A disconnected graph **cannot** have a spanning tree (though it can have a spanning *forest*).

---

> **Exam Tip:** In short answer questions, you are frequently asked to define the structural constraints of an MST. The holy trinity to write down is: **(1) Contains all vertices ($V$), (2) Contains $V-1$ edges, (3) Contains zero cycles.** Also, memorize $N^{N-2}$ (Cayley's Formula) as it is a highly common multiple-choice question!
