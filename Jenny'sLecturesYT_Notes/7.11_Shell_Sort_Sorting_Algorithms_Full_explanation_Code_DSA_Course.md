## 🔑 Shell Sort = Improved Insertion Sort with diminishing gaps

## 📐 Algorithm
1. Choose gap sequence (typically n/2, n/4, ..., 1)
2. For each gap, perform **gapped insertion sort**
3. Final pass with gap=1 is regular insertion sort (but nearly sorted!)

## 💻 C Code
```c
void shellSort(int arr[], int n) {
    for (int gap = n/2; gap > 0; gap /= 2) {
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j;
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap)
                arr[j] = arr[j - gap];
            arr[j] = temp;
        }
    }
}
```

## 📐 Trace: [12, 34, 54, 2, 3] with gaps 2, 1
```
Gap=2: Compare elements 2 apart → [3, 2, 12, 34, 54] (partially sorted)
Gap=1: Regular insertion sort → [2, 3, 12, 34, 54] ✅
```

## ⏱️ Time: O(n²) worst, **O(n^1.5)** with good gaps | Space: O(1) | Stable: ❌
