## 📌 Executive Summary

This lecture covers array operations: **traversal** (visiting every element) and **insertion** (at beginning, end, or specific position). It includes complete C code for each operation, discusses bound checking, and analyzes time complexity of insertion in sorted vs unsorted arrays.

## 🔑 Array Operations Overview
1. **Traversal** — visit and print every element
2. **Insertion** — add element at beginning, end, or specific position
3. **Deletion** — remove element (covered in next lecture)
4. **Searching** — find if element exists
5. **Sorting** — arrange elements in order

## 💻 Traversal

```c
#include <stdio.h>
int main() {
    int a[50], size, i;

    printf("Enter size of array: ");
    scanf("%d", &size);

    // Bound checking (no auto bound check in C)
    if (size > 50) {
        printf("Overflow condition!");
        return 1;
    }

    // Runtime initialization
    printf("Enter elements of array:\n");
    for (i = 0; i < size; i++)
        scanf("%d", &a[i]);

    // Traversal — print all elements
    printf("Elements in array:\n");
    for (i = 0; i < size; i++)
        printf("%d ", a[i]);

    return 0;
}
```

**Time Complexity: O(n)** — visit each element exactly once.

## 💻 Insertion at a Specific Position

### Algorithm
1. Check if array is full (overflow condition)
2. Validate position (1 ≤ position ≤ size+1)
3. **Shift elements** from `size-1` down to `position-1` **one step right**
4. Place new element at `position-1`
5. Increment size

### Visual Example
```
Before: [6, 2, 0, 4, 5]    Insert 10 at position 3
         0  1  2  3  4

Step 1: Shift right from end
        [6, 2, 0, 0, 4, 5]

Step 2: Insert 10 at index 2
        [6, 2, 10, 0, 4, 5]
```

### C Code
```c
printf("Enter data to insert: ");
scanf("%d", &num);
printf("Enter position: ");
scanf("%d", &pos);

if (pos <= 0 || pos > size + 1) {
    printf("Invalid position!");
} else {
    // Shift elements right
    for (i = size - 1; i >= pos - 1; i--)
        a[i + 1] = a[i];

    a[pos - 1] = num;   // Insert element
    size++;              // Update size
}
```

## 💻 Insertion at Beginning

```c
for (i = size - 1; i >= 0; i--)
    a[i + 1] = a[i];      // Shift ALL elements right
a[0] = num;
size++;
```

## 💻 Insertion at End

```c
a[size] = num;    // Simply place at end — no shifting needed
size++;
```

## ⏱️ Time Complexity

| Operation | Best Case | Worst Case |
|-----------|-----------|------------|
| Insert at **end** | O(1) | O(1) |
| Insert at **beginning** | O(n) | O(n) |
| Insert at **position p** | O(1) | O(n) |
| General | Θ(n - p) where p = position | |

### Unsorted Array Optimization — O(1)
```c
a[size] = a[pos - 1];   // Save displaced element at end
a[pos - 1] = num;       // Insert new element
size++;
// Time: O(1) — no shifting needed!
```

> ⚠️ This only works for **unsorted** arrays. For **sorted** arrays, you must shift → O(n).

---

## 💡 Key Takeaways

- C arrays have **no automatic bound checking** — programmer must handle it
- Insertion requires **shifting elements** to make room (except at end)
- Worst case insertion: O(n) when inserting at beginning
- Best algorithm for unsorted array insertion: **O(1)** (swap trick)
