## 📌 Executive Summary
Reversing a Circular Linked List (CLL) is fundamentally similar to reversing a standard Singly Linked List. Because we only have forward links (`next`), we must use the **3-pointer sliding window technique** (`previous`, `current`, `nextnode`). 

The primary difference lies in the loop termination condition and the **final step**: after reversing all the intermediate links, you must manually reconnect the new last node back to the new first node to maintain the circular structure.

---

## 🛠️ The 3-Pointer Algorithm

If we are maintaining only a `tail` pointer, `tail->next` gives us the first node.

### Step 1: Initialization
We initialize our pointers to anchor the start of the list:
- `current = tail->next;` (Points to the first node)
- `nextnode = current->next;` (Points to the second node)
- `previous` is left uninitialized for now.

### Step 2: The Reversal Loop
We slide the three pointers forward, reversing the links behind us. We stop when `current` reaches the `tail` node.
```c
while (current != tail) {
    previous = current;
    current = nextnode;
    nextnode = current->next;  // Secure the path forward
    current->next = previous;  // Reverse the link
}
```

### Step 3: Fixing the Circular Link (The Final Swap)
When the loop finishes:
- `current` is pointing to the old `tail` (which is now the new first node).
- `nextnode` is pointing to the old first node (which should now be the new `tail`).
- The link between the old first node and the old `tail` has NOT been reversed yet.

We must manually update the old first node to point to the new first node, and shift the global `tail` pointer:
- `nextnode->next = tail;`
- `tail = nextnode;`

## 💻 Full C Code Implementation

```c
void reverseCLL() {
    struct node *current, *previous, *nextnode;
    
    // Check if list is empty
    if (tail == NULL) {
        printf("List is empty.\n");
        return;
    }
    
    // Edge case: Only 1 node (Reversing a 1-node circle does nothing)
    if (tail->next == tail) {
        return; 
    }
    
    // ----------------------------------------
    // 1. Initialize Pointers
    // ----------------------------------------
    current = tail->next;
    nextnode = current->next;
    
    // ----------------------------------------
    // 2. The Sliding Window Loop
    // ----------------------------------------
    // Stop when current reaches the old tail
    while (current != tail) {
        previous = current;
        current = nextnode;
        
        nextnode = current->next; // Look ahead
        current->next = previous; // Reverse link behind
    }
    
    // ----------------------------------------
    // 3. Secure the Circular Link
    // ----------------------------------------
    // nextnode is currently the old first node. It needs to point to 
    // the new first node (which is the old tail).
    nextnode->next = tail;
    
    // Shift the global tail pointer to the old first node
    tail = nextnode;
}
```

> [!WARNING]
> **Exam Trap (Infinite Loop during Reversal):**
> A common mistake is writing `while(current != NULL)`. Since a Circular Linked List has no `NULL` pointers, this loop will run forever, constantly re-reversing the list! You **must** set the termination condition to `while(current != tail)` so the loop breaks exactly when the sliding window reaches the end of the original list.
