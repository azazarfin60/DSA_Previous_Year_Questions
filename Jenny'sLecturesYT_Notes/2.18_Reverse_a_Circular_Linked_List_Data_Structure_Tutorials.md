## 📌 Executive Summary

This lecture covers reversing a circular linked list using a modified iterative approach with three pointers, using **do-while** instead of while loop.

---

## 💻 Algorithm
```c
void reverse() {
    if (head == NULL || head->next == head) return;

    struct node *prev = NULL;
    struct node *current = head;
    struct node *next;

    do {
        next = current->next;       // Save next
        current->next = prev;       // Reverse link
        prev = current;             // Advance prev
        current = next;             // Advance current
    } while (current != head);

    head->next = prev;     // Old head → new last, point to new head
    head = prev;           // Update head
}
```

---

## 🔄 Trace
```
Before: [5|200] → [6|300] → [8|100] ↺  (head @ 100)

Iter 1: next=200, [5|NULL], prev=100, curr=200
Iter 2: next=300, [6|100], prev=200, curr=300
Iter 3: next=100, [8|200], prev=300, curr=100 → curr==head → STOP

head->next = prev → [5|300]
head = prev → head = 300

After: [8|200] → [6|100] → [5|300] ↺  (head @ 300)  ✅
```

---

## ⏱️ Complexity
- **Time: O(n)**, **Space: O(1)**

---

## 💡 Key Takeaways

- Use **do-while** (current != head) — same as CLL traversal
- After loop: set `head->next = prev` to maintain circular link
- Update `head = prev` (new first node)
