## 📌 Executive Summary

This lecture explains how to convert **Infix to Prefix** using a stack. The method is: reverse the infix, apply the postfix algorithm (with modified rules), then reverse the result.

---

## 📐 Algorithm

1. **Reverse** the infix expression (swap `(` with `)` and vice versa)
2. Apply the **infix-to-postfix** algorithm on the reversed expression
   - Only difference: for same precedence, **don't pop** (push instead)
3. **Reverse** the result → this is the prefix expression

---

## 📐 Example: `A + B * C`

### Step 1: Reverse
```
Original:  A + B * C
Reversed:  C * B + A
```

### Step 2: Apply Postfix Algorithm

| Symbol | Stack | Output |
|--------|-------|--------|
| C | | C |
| * | * | C |
| B | * | C B |
| + | + | C B * |
| A | + | C B * A |
| End | | C B * A + |

### Step 3: Reverse Result
```
C B * A +  →  + A * B C
```

> **Prefix: `+ A * B C`** ✅

---

## 📐 Example with Parentheses: `(A + B) * C - D`

### Step 1: Reverse (swap brackets)
```
D - C * (B + A)
```

### Step 2: Postfix on reversed

| Symbol | Stack | Output |
|--------|-------|--------|
| D | | D |
| - | - | D |
| C | - | D C |
| * | - * | D C |
| ( | - * ( | D C |
| B | - * ( | D C B |
| + | - * ( + | D C B |
| A | - * ( + | D C B A |
| ) | - * | D C B A + |
| End | | D C B A + * - |

### Step 3: Reverse
```
D C B A + * -  →  - * + A B C D
```

> **Prefix: `- * + A B C D`** ✅

---

## 💡 Key Takeaways

- Infix → Prefix: **Reverse → Apply Postfix → Reverse**
- When applying postfix on reversed: for **same precedence, push** (don't pop)
- Parentheses get swapped during reversal
- This is a frequently asked exam question
