# 9.2 Huffman Coding with Probabilities | UGC NET Previous Year Question

## 📐 Example with Probabilities

Given: A=0.4, B=0.2, C=0.2, D=0.1, E=0.1

### Step-by-Step Construction
```
Step 1: Pick two smallest: D(0.1) + E(0.1) = DE(0.2)
        Remaining: {A:0.4, B:0.2, C:0.2, DE:0.2}

Step 2: Pick two smallest: C(0.2) + DE(0.2) = CDE(0.4)
        Remaining: {A:0.4, B:0.2, CDE:0.4}

Step 3: Pick two smallest: B(0.2) + A(0.4) = BA(0.6)
        Remaining: {BA:0.6, CDE:0.4}

Step 4: Pick two smallest: CDE(0.4) + BA(0.6) = Root(1.0)
```

### Final Huffman Tree
```
         1.0
        /    \
     CDE:0.4  BA:0.6
     /    \    /   \
   C:0.2 DE:0.2 B:0.2 A:0.4
          / \
       D:0.1 E:0.1
```

### Codes (left=0, right=1)
| Symbol | Code | Length |
|--------|------|--------|
| C | 00 | 2 |
| D | 010 | 3 |
| E | 011 | 3 |
| B | 10 | 2 |
| A | 11 | 2 |

### Average Code Length
```
= 0.4(2) + 0.2(2) + 0.2(2) + 0.1(3) + 0.1(3)
= 0.8 + 0.4 + 0.4 + 0.3 + 0.3
= 2.2 bits
```
