## 🔑 Why Circular Queue?
> In linear queue, after dequeue, front positions waste space. Circular queue wraps around using `% MAX`.

## 💻 C Implementation
```c
#define MAX 5
int cq[MAX], front = -1, rear = -1;

int isFull() { return (rear + 1) % MAX == front; }
int isEmpty() { return front == -1; }

void enqueue(int data) {
    if (isFull()) { printf("Overflow!\n"); return; }
    if (isEmpty()) front = 0;
    rear = (rear + 1) % MAX;
    cq[rear] = data;
}

int dequeue() {
    if (isEmpty()) { printf("Underflow!\n"); return -1; }
    int data = cq[front];
    if (front == rear)  // Last element
        front = rear = -1;
    else
        front = (front + 1) % MAX;
    return data;
}
```

## 💡 Key formula: `next = (current + 1) % MAX` wraps around the array.
