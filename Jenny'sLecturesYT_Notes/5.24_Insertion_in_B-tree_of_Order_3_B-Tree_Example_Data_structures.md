## 📐 Order 3 (2-3 Tree): max 2 keys, max 3 children, min 1 key

### Example: Insert 8, 5, 1, 7, 3, 12, 9, 6

```
Insert 8:    [8]
Insert 5:    [5, 8]  (full)
Insert 1:    Overflow! Split: middle=5 up
                [5]
               /   \
             [1]   [8]
Insert 7:     [5]
             /   \
           [1]   [7, 8]
Insert 3:    [5]
            /   \
         [1, 3]  [7, 8]
Insert 12: Overflow in right! middle=8 up
              [5, 8]
             / |  \
          [1,3] [7] [12]
Insert 9:    [5, 8]
            / |   \
        [1,3] [7] [9, 12]
Insert 6:  6 is between 5 and 8, goes to middle child [7] → [6, 7]
            [5, 8]
           / |   \
       [1,3] [6,7] [9, 12]
```

## 💡 Order 3 splits at 2nd element (middle of 3).
