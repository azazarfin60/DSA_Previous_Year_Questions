# 6.12 Bridges (Cut Edge) in a Graph | Find All Bridges in a Graph | Graph Theory

## 🔑 Definition
> A **bridge** is an edge whose removal **disconnects** the graph (increases connected components).

## 📐 Algorithm (Tarjan's)
Use DFS with discovery time and low values:
- **disc[u]**: when u was discovered
- **low[u]**: lowest disc reachable from subtree of u

> Edge (u, v) is a bridge if: **low[v] > disc[u]**

```c
int disc[MAX], low[MAX], timer = 0;

void dfs(int u, int parent) {
    disc[u] = low[u] = timer++;
    for each neighbor v of u:
        if (disc[v] == -1) {       // Unvisited
            dfs(v, u);
            low[u] = min(low[u], low[v]);
            if (low[v] > disc[u])
                printf("Bridge: %d-%d\n", u, v);
        } else if (v != parent) {
            low[u] = min(low[u], disc[v]);
        }
}
```

## ⏱️ Time: O(V + E) — single DFS pass
