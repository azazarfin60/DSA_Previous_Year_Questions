## 💻 C Implementation
```c
#define MAX 100
int queue[MAX], front = -1, rear = -1;

void enqueue(int data) {
    if (rear == MAX - 1) { printf("Overflow!\n"); return; }
    if (front == -1) front = 0;
    queue[++rear] = data;
}

int dequeue() {
    if (front == -1 || front > rear) { printf("Underflow!\n"); return -1; }
    return queue[front++];
}

void display() {
    for (int i = front; i <= rear; i++)
        printf("%d ", queue[i]);
}
```

## ⚠️ Problem: After dequeue, front positions are wasted. Solution: **Circular Queue**.
