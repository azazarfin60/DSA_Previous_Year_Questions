## 📐 Complex Example: `K + L - M * N + (O ^ P) * W / U / V * T + Q`

This video provides a **full step-by-step trace** of a complex expression to reinforce the algorithm.

### Rules Recap:
1. Operand → output
2. `(` → push
3. `)` → pop until `(`
4. Operator → pop while top ≥ precedence, then push
5. `^` → right-to-left (don't pop equal precedence `^`)
6. End → pop all

**Result: `K L + M N * - O P ^ W * U / V / T * + Q +`**
