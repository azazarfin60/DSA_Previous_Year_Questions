## 📌 Executive Summary

This lecture explains the deep relationship between **pointers and arrays** in C — how array names decay to pointers, pointer arithmetic on arrays, and accessing elements via pointer notation.

---

## 🔑 Array Name = Base Address

```c
int arr[5] = {10, 20, 30, 40, 50};
// arr == &arr[0]  (base address)
// *arr == arr[0] == 10
```

> The array name `arr` is a **constant pointer** to the first element.

---

## 📐 Pointer Arithmetic on Arrays

```c
int *p = arr;    // p points to arr[0]

p + 0 → &arr[0]    *(p + 0) = 10
p + 1 → &arr[1]    *(p + 1) = 20
p + 2 → &arr[2]    *(p + 2) = 30
p + i → &arr[i]    *(p + i) = arr[i]
```

### Memory Layout (4 bytes per int):
```
Address:  1000  1004  1008  1012  1016
Value:     10    20    30    40    50
Index:    [0]   [1]   [2]   [3]   [4]
Pointer:   p    p+1   p+2   p+3   p+4
```

> `p + 1` moves by `sizeof(int)` = 4 bytes, NOT 1 byte!

---

## 📐 Equivalent Notations

| Expression | Meaning | Value |
|-----------|---------|-------|
| `arr[i]` | i-th element | 30 (i=2) |
| `*(arr + i)` | Same thing | 30 |
| `*(p + i)` | Same thing | 30 |
| `p[i]` | Same thing | 30 |

> **All four are equivalent!** `arr[i]` internally becomes `*(arr + i)`.

---

## 💻 Traversing Array with Pointer

```c
void printArray(int *p, int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", *(p + i));   // or p[i]
    }
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    printArray(arr, 5);   // arr decays to pointer
}
```

---

## ⚠️ Key Differences: Array vs Pointer

| Feature | Array | Pointer |
|---------|-------|---------|
| `sizeof` | Total array size | Pointer size (4/8 bytes) |
| `&` operator | Address of whole array | Address of pointer variable |
| Modifiable | ❌ `arr++` illegal | ✅ `p++` legal |
| Storage | Stack (fixed) | Can point anywhere |

---

## 💡 Key Takeaways

- Array name = **constant pointer** to first element
- `arr[i]` ≡ `*(arr + i)` ≡ `*(p + i)` ≡ `p[i]`
- Pointer arithmetic: `p + i` adds `i × sizeof(type)` bytes
- Arrays decay to pointers when passed to functions
- Cannot increment array name (`arr++`), but can increment pointer (`p++`)
