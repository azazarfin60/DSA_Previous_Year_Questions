## 📌 Executive Summary

This lecture introduces the **Circular Linked List (CLL)** — a variation where the **last node's next points back to the first node** instead of NULL, forming a continuous loop.

---

## 🔑 CLL vs SLL
```
SLL:  Head → [5|200] → [6|300] → [8|NULL]     (ends at NULL)
CLL:  Head → [5|200] → [6|300] → [8|100] ↺    (loops back)
```

---

## 💻 Node Structure (Same as SLL)
```c
struct node {
    int data;
    struct node *next;
};
struct node *head = NULL;
```

---

## 💻 Creation
```c
void createCLL(int n) {
    struct node *newNode, *temp;
    for (int i = 0; i < n; i++) {
        newNode = (struct node *)malloc(sizeof(struct node));
        scanf("%d", &newNode->data);

        if (head == NULL) {
            head = newNode;
            newNode->next = head;    // Points to itself
        } else {
            temp = head;
            while (temp->next != head)
                temp = temp->next;
            temp->next = newNode;
            newNode->next = head;    // New last → head
        }
    }
}
```

---

## 💻 Display
```c
void display() {
    if (head == NULL) return;
    struct node *temp = head;
    do {
        printf("%d → ", temp->data);
        temp = temp->next;
    } while (temp != head);    // Stop when back at head
}
```

> ⚠️ Use **do-while** (not while) — because `temp != head` is false at start!

---

## 💡 Key Takeaways

- Last node's `next = head` (not NULL)
- Use **do-while** for traversal
- Termination: `temp->next != head` (not `temp != NULL`)
- Single-node CLL: `head->next = head` (points to itself)
- Use cases: round-robin scheduling, circular buffers
