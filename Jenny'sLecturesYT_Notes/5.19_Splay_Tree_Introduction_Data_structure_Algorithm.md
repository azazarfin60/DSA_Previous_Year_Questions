## 🔑 Splay Tree: Self-adjusting BST
> Moves most recently accessed node to root via **splaying**. No balance factor stored. Amortized O(log n).

## 📐 Three Splay Cases
| Case | Condition | Rotation |
|------|-----------|----------|
| **Zig** | Node is child of root | Single rotation |
| **Zig-Zig** | Node & parent same direction | Two same rotations |
| **Zig-Zag** | Node & parent opposite direction | Two opposite rotations |

## 🎯 Key Advantage: Frequently accessed nodes stay near root → locality of reference.
