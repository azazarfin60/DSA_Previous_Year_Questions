# 5.8 Construct Binary Tree from Postorder and Inorder with example | Data structures Course

📺 [Watch on YouTube](https://www.youtube.com/watch?v=s5XRtcud35E)

---

## 📌 Executive Summary

This lecture shows constructing a binary tree from **Postorder and Inorder** traversals. The key difference from preorder: the root is the **last** element of postorder.

---

## 📐 Algorithm

1. **Last element of Postorder** = Root
2. Find root in Inorder → split left/right
3. Recursively repeat (process **right subtree first** in postorder)

---

## 📐 Example

```
Inorder:   D B E A F C
Postorder: D E B F C A
```

### Step 1: Root = **A** (last of postorder)
```
Inorder: [D B E] A [F C]
```

### Step 2: Right subtree — Postorder ends at **C**
```
Inorder: [F] C [ ]
```

### Step 3: Left subtree — Postorder ends at **B**
```
Inorder: [D] B [E]
```

### Result:
```
       A
      / \
     B   C
    / \ /
   D  E F
```

---

## ⚠️ Important Note
When splitting postorder for subtrees:
- Count elements in left subtree from inorder → take that many from postorder (from left)
- Remaining elements (before root) → right subtree

---

## 💡 Key Takeaways

- Postorder root is **last** (vs Preorder root is first)
- Same splitting logic using Inorder
- Process: pick root from end → split inorder → recurse
