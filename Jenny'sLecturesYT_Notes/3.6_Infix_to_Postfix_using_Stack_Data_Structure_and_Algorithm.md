## 📐 Example with Parentheses: `(A + B) * (C - D)`

| Symbol | Stack | Output |
|--------|-------|--------|
| ( | ( | |
| A | ( | A |
| + | ( + | A |
| B | ( + | A B |
| ) | | A B + |
| * | * | A B + |
| ( | * ( | A B + |
| C | * ( | A B + C |
| - | * ( - | A B + C |
| D | * ( - | A B + C D |
| ) | * | A B + C D - |
| End | | A B + C D - * |

**Postfix: `A B + C D - *`** ✅

---

## 📐 Example with `^`: `A + B ^ C * D`

| Symbol | Stack | Output |
|--------|-------|--------|
| A | | A |
| + | + | A |
| B | + | A B |
| ^ | + ^ | A B |
| C | + ^ | A B C |
| * | + * | A B C ^ |
| D | + * | A B C ^ D |
| End | | A B C ^ D * + |

**Postfix: `A B C ^ D * +`** ✅

## 💡 Key: `^` has highest precedence and right-to-left associativity.
