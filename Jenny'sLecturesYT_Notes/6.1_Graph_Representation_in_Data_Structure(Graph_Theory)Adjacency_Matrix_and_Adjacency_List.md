## 📌 Executive Summary

This lecture covers graph basics — types of graphs and the **two main representations**: Adjacency Matrix and Adjacency List, with their space/time trade-offs.

---

## 🔑 Graph Definition
> G = (V, E) where V = set of vertices, E = set of edges

### Types of Graphs
| Type | Description |
|------|-----------|
| **Undirected** | Edges have no direction (A—B) |
| **Directed** | Edges have direction (A→B) |
| **Weighted** | Edges have weights/costs |
| **Unweighted** | All edges equal |

---

## 📐 Adjacency Matrix

A 2D array of size V×V where `adj[i][j] = 1` if edge exists between i and j.

### Undirected Graph Example
```
Graph:  0 — 1       Adjacency Matrix:
        |   |         0  1  2  3
        3 — 2       0 [0, 1, 0, 1]
                    1 [1, 0, 1, 0]
                    2 [0, 1, 0, 1]
                    3 [1, 0, 1, 0]
```
> Undirected → matrix is **symmetric** (adj[i][j] = adj[j][i])

### Directed Graph
```
0 → 1, 0 → 3, 1 → 2, 3 → 2

    0  1  2  3
0 [ 0, 1, 0, 1]     NOT symmetric!
1 [ 0, 0, 1, 0]
2 [ 0, 0, 0, 0]
3 [ 0, 0, 1, 0]
```

### Weighted Graph
Store **weight** instead of 1: `adj[i][j] = weight`

---

## 📐 Adjacency List

Array of linked lists. Each index stores a list of its neighbors.

```
Graph: 0—1, 0—3, 1—2, 3—2

0 → [1] → [3]
1 → [0] → [2]
2 → [1] → [3]
3 → [0] → [2]
```

---

## 📊 Comparison

| Aspect | Adjacency Matrix | Adjacency List |
|--------|-----------------|----------------|
| Space | **O(V²)** | **O(V + E)** |
| Check edge (i,j) | **O(1)** | O(degree) |
| Find all neighbors | O(V) | **O(degree)** |
| Best for | **Dense** graphs | **Sparse** graphs |
| Add edge | O(1) | O(1) |
| Remove edge | O(1) | O(degree) |

---

## 💻 C Implementation

### Adjacency Matrix
```c
int adj[MAX][MAX] = {0};

void addEdge(int u, int v) {
    adj[u][v] = 1;
    adj[v][u] = 1;  // Undirected
}
```

### Adjacency List
```c
struct node {
    int vertex;
    struct node *next;
};
struct node *adj[MAX] = {NULL};
```

---

## 💡 Key Takeaways

- **Matrix**: O(V²) space, O(1) edge lookup — best for dense graphs
- **List**: O(V+E) space, efficient neighbor traversal — best for sparse
- Undirected matrix = **symmetric**
- Degree of vertex = row sum (matrix) or list length (list)
