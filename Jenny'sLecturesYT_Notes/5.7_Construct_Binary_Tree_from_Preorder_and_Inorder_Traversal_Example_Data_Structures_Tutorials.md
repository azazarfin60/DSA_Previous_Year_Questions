# 5.7 Construct Binary Tree from Preorder and Inorder Traversal | Example | Data Structures Tutorials

📺 [Watch on YouTube](https://www.youtube.com/watch?v=PoBGyrIWisE)

---

## 📌 Executive Summary

This lecture shows how to **reconstruct a unique binary tree** given its Preorder and Inorder traversals — a very frequently asked exam question.

---

## 📐 Algorithm

1. **First element of Preorder** = Root of tree
2. Find that root in **Inorder** → elements to **left** = left subtree, elements to **right** = right subtree
3. **Recursively** apply for left and right subtrees

---

## 📐 Example

```
Inorder:  D B E A F C
Preorder: A B D E C F
```

### Step 1: Root = A (first of preorder)
```
Inorder split:  [D B E]  A  [F C]
                  left       right
```

### Step 2: Left subtree — Preorder: B D E
Root = **B**, Inorder split: [D] B [E]

### Step 3: Right subtree — Preorder: C F
Root = **C**, Inorder split: [F] C [ ]

### Result:
```
       A
      / \
     B   C
    / \ /
   D  E F
```

### Verification:
- Inorder (L-Root-R): D B E A F C ✅
- Preorder (Root-L-R): A B D E C F ✅

---

## 📐 Key Rules

| Traversal | Root Position |
|-----------|--------------|
| Preorder | **First** element |
| Postorder | **Last** element |
| Inorder | **Splitter** — tells left vs right |

> **Inorder is essential** — it determines which nodes go to left vs right subtree.

---

## 💡 Key Takeaways

- Preorder gives **root**, Inorder **splits** left/right
- Process: pick root → split inorder → recurse
- Works only with **Inorder + one other** traversal
- **Time: O(n²)** naive, O(n) with hash map
- Practice step-by-step — guaranteed exam question!
