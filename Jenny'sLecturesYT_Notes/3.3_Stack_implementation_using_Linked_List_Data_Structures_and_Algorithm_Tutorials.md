## 📌 Executive Summary

This lecture covers implementing a **Stack using a Linked List** — push and pop operations by inserting/deleting at the **beginning** of the list (O(1) for both).

---

## 💻 Implementation
```c
struct node {
    int data;
    struct node *next;
};
struct node *top = NULL;   // top of stack

void push(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;
    newNode->next = top;   // New → old top
    top = newNode;         // Update top
}

int pop() {
    if (top == NULL) {
        printf("Stack Underflow!\n");
        return -1;
    }
    struct node *temp = top;
    int data = temp->data;
    top = top->next;       // Move top down
    free(temp);
    return data;
}

int peek() {
    if (top == NULL) return -1;
    return top->data;
}

void display() {
    struct node *temp = top;
    while (temp != NULL) {
        printf("%d\n", temp->data);
        temp = temp->next;
    }
}
```

## 📊 Array vs Linked List Stack

| Feature | Array | Linked List |
|---------|-------|-------------|
| Size | Fixed | Dynamic |
| Overflow | Possible | Not possible* |
| Memory | Contiguous | Scattered + pointer |
| Push/Pop | O(1) | O(1) |

*Until system memory runs out

---

## 💡 Key Takeaways

- Push = **insert at beginning** of linked list
- Pop = **delete from beginning** of linked list
- Both are O(1) — no traversal needed
- No overflow issue — dynamic memory allocation
- `top` replaces `head` as the pointer name
