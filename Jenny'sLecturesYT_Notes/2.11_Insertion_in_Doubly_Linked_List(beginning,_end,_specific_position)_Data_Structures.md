## 📌 Executive Summary

This lecture covers **three insertion operations** in a Doubly Linked List. DLL insertion requires updating **up to 4 pointers** compared to 2 in SLL.

---

## 💻 1. Insert at Beginning
```c
void insertAtBeginning(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;
    newNode->prev = NULL;

    if (head == NULL) {
        newNode->next = NULL;
        head = newNode;
    } else {
        newNode->next = head;    // New → old head
        head->prev = newNode;    // Old head's prev → new
        head = newNode;          // Update head
    }
}
```
**Time: O(1)** — 3 pointer updates

---

## 💻 2. Insert at End
```c
void insertAtEnd(int data) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;
    newNode->next = NULL;

    if (head == NULL) {
        newNode->prev = NULL;
        head = newNode;
    } else {
        struct node *temp = head;
        while (temp->next != NULL)
            temp = temp->next;
        temp->next = newNode;     // Last → new
        newNode->prev = temp;     // New's prev → last
    }
}
```
**Time: O(n)**

---

## 💻 3. Insert After Position
```c
void insertAfterPos(int data, int pos) {
    struct node *newNode = (struct node *)malloc(sizeof(struct node));
    newNode->data = data;

    struct node *temp = head;
    for (int i = 1; i < pos; i++)
        temp = temp->next;

    // Update 4 pointers:
    newNode->next = temp->next;         // 1. New → right node
    if (temp->next != NULL)
        temp->next->prev = newNode;     // 2. Right node → new
    temp->next = newNode;               // 3. Left node → new
    newNode->prev = temp;               // 4. New → left node
}
```
**Time: O(n)**

### Visual (Insert after position 2)
```
Before: [NULL|5|200] ⇄ [100|6|300] ⇄ [200|8|NULL]

After:  [NULL|5|200] ⇄ [100|6|400] ⇄ [200|10|300] ⇄ [400|8|NULL]
                                       ↑ new node @400
```

---

## 💡 Key Takeaways

- DLL insertion updates up to **4 pointers** (vs 2 in SLL)
- Same "right link first" rule applies
- Check if `temp->next != NULL` before updating `temp->next->prev`
