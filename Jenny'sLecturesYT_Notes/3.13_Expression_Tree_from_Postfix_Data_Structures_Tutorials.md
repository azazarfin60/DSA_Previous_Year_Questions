## 📐 Algorithm: Build Expression Tree from Postfix

1. Read postfix **left to right**
2. **Operand** → create leaf node, push to stack
3. **Operator** → pop two nodes, create new node with operator, right child = first pop, left child = second pop, push new node

### Example: `A B + C D - *`

```
Read A → push leaf(A)
Read B → push leaf(B)
Read + → pop B,A → create (+, left=A, right=B) → push

Read C → push leaf(C)
Read D → push leaf(D)
Read - → pop D,C → create (-, left=C, right=D) → push

Read * → pop (-node),(+node) → create (*, left=+node, right=-node)

Final tree:
       *
      / \
     +   -
    / \ / \
   A  B C  D
```

## 💡 Stack holds tree **nodes** (not just values) during construction.
