## 📌 Executive Summary

This lecture covers **three deletion operations** from a Doubly Linked List, with the key advantage that DLL deletion can directly access the previous node via `prev` pointer.

---

## 💻 1. Delete from Beginning
```c
void deleteFromBeginning() {
    if (head == NULL) { printf("Empty!\n"); return; }
    struct node *temp = head;
    head = head->next;
    if (head != NULL)
        head->prev = NULL;    // New head has no previous
    free(temp);
}
```
**Time: O(1)**

---

## 💻 2. Delete from End
```c
void deleteFromEnd() {
    if (head == NULL) { printf("Empty!\n"); return; }
    struct node *temp = head;
    if (temp->next == NULL) {    // Single node
        head = NULL;
        free(temp);
        return;
    }
    while (temp->next != NULL)
        temp = temp->next;       // Go to last
    temp->prev->next = NULL;     // Second-last's next = NULL
    free(temp);
}
```
**Time: O(n)**

---

## 💻 3. Delete from Position
```c
void deleteFromPos(int pos) {
    if (head == NULL) { printf("Empty!\n"); return; }
    struct node *temp = head;

    if (pos == 1) { deleteFromBeginning(); return; }

    for (int i = 1; i < pos; i++)
        temp = temp->next;

    // Bypass temp
    if (temp->prev != NULL)
        temp->prev->next = temp->next;
    if (temp->next != NULL)
        temp->next->prev = temp->prev;
    free(temp);
}
```
**Time: O(n)**

---

## 🎯 DLL Deletion Advantage
> In SLL, to delete a node you need the **previous node** — requires traversal or two pointers.  
> In DLL, `temp->prev` gives **direct access** to previous node!

---

## 💡 Key Takeaways

- Handle **empty list** and **single node** as special cases
- Always `free()` deleted nodes
- DLL advantage: direct prev access → simpler deletion logic
