## 📌 Executive Summary

Counting Sort is a **non-comparison** integer sorting algorithm. It counts occurrences of each value, then places elements in correct positions. Efficient when range of input (k) is not significantly larger than n.

---

## 📐 Algorithm

1. Find **max** value in array → determine count array size (k+1)
2. **Count** occurrences of each value
3. **Cumulative count** — transform count to positions
4. **Place elements** in output array using cumulative count (scan right to left for stability)

---

## 💻 C Code

```c
void countingSort(int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++)
        if (arr[i] > max) max = arr[i];

    int count[max + 1];
    int output[n];

    // Step 1: Initialize count array
    for (int i = 0; i <= max; i++) count[i] = 0;

    // Step 2: Count occurrences
    for (int i = 0; i < n; i++) count[arr[i]]++;

    // Step 3: Cumulative count
    for (int i = 1; i <= max; i++) count[i] += count[i - 1];

    // Step 4: Place elements (right to left for stability)
    for (int i = n - 1; i >= 0; i--) {
        output[count[arr[i]] - 1] = arr[i];
        count[arr[i]]--;
    }

    // Copy back
    for (int i = 0; i < n; i++) arr[i] = output[i];
}
```

## 📐 Trace: Sort [4, 2, 2, 8, 3, 3, 1]

```

  Count: 0  1  3  5  6  6  6  6  7

  arr[0]=4 → output[5] = 4

```

## ⏱️ Complexity

| Metric | Value |
|--------|-------|
| **Time** | **O(n + k)** |
| **Space** | **O(n + k)** |

Where k = range of input values (max - min + 1)

---

## 📊 Properties

| Property | Value |
|----------|-------|
| Comparison-based | ❌ No |
| In-place | ❌ No |
| Stable | ✅ Yes (with right-to-left placement) |
| Best when | k ≈ n (range close to count) |
| Bad when | k >> n (huge range, few elements) |

---

## 💡 Key Takeaways

- Counts occurrences → cumulative sum → place directly
- **O(n + k)** — faster than comparison sorts when k is small
- **Stable** — maintains relative order of equal elements
- Used as **subroutine in Radix Sort**
- Not suitable for floating point or very large ranges
