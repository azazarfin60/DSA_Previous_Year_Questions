# 8.13 Minimum Spanning Tree (MST)

> **Exam Frequency:** 🔴 Very High Frequency | **Typical Marks:** 02–04 | **Section:** B  
> **Course Outcome:** Maps to CO4

---

## 1. Core Definition
In a **weighted graph**, every edge has a specific numeric weight (representing cost, distance, time, etc.). 

A **Minimum Spanning Tree (MST)** is a spanning tree of a weighted, connected, undirected graph that has the **minimum possible total edge weight** among all valid spanning trees for that graph.

---

## 2. Real-World Applications
Why do we care about finding the MST?
- **Telecommunications:** Laying down physical fiber optic cables to connect multiple cities so that every city is connected, but the total length of cable used is as short as possible.
- **Circuit Design:** Wiring electrical pins together using the minimum amount of wire.
- **Water/Gas Networks:** Constructing a pipeline network connecting all houses with minimum pipe length.

---

## 3. Key Properties
1. It is a Spanning Tree, so it still has exactly **$V$ vertices** and **$V - 1$ edges**.
2. It contains **no cycles**.
3. **Uniqueness:** If all edge weights in a graph are *unique* (no two edges have the same weight), then there is exactly **one unique MST**.
4. If some edge weights are identical, there may be *multiple* valid MSTs, but they will all have the exact same *total minimum weight*.

---

## 4. How to find the MST?
Finding the MST by manually checking all possible spanning trees is computationally impossible for large graphs (O($V^{(V-2)}$) time).

Instead, we use efficient **Greedy Algorithms** that find the optimal MST in polynomial time. The two most famous are:
1. **Prim's Algorithm** (Grows a single tree from a starting node).
2. **Kruskal's Algorithm** (Sorts edges and merges them, avoiding cycles).

---

## 5. Exam Strategy & Practice Questions

### Typical Exam Questions:
1. **"What is a Minimum Spanning Tree? Give two real-world applications." (03 marks)**
   - *Strategy:* Define a spanning tree, then add the condition "minimum total edge weight". Mention telecommunications (fiber optic layouts) and pipeline networks.
2. **"Does every graph have a unique MST?" (02 marks)**
   - *Strategy:* Answer NO. If edge weights are not unique, multiple MSTs can exist that share the same minimum total cost.

### Practice Questions:
1. If a graph has $V=6$ and $E=10$, how many edges will the Minimum Spanning Tree contain? *(Answer: Exactly 5 edges).*
