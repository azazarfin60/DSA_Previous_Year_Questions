# 9.5 Master Theorem

> **Exam Frequency:** 🟢 Moderate Frequency | **Typical Marks:** 04–05 | **Section:** B  
> **Course Outcome:** Maps to CO5

---

## 1. Core Definition
The **Master Theorem** provides a direct, "shortcut" mathematical formula to find the Big-O time complexity of recurrence relations. Unlike the Substitution Method, you don't need to expand equations manually.

It applies to recurrences of the standard form:
> **$T(n) = a T(n/b) + f(n)$**

**Where:**
- $n$: Size of the input
- $a$: Number of subproblems in the recursion ($a \ge 1$)
- $b$: The factor by which the subproblem size is divided ($b > 1$)
- $f(n)$: The cost of the work done outside the recursive calls (e.g., dividing and merging). It is usually in the form $O(n^k)$.

---

## 2. The Three Cases of Master Theorem

To solve the relation, you must compare two values:
1. The external work: **$f(n)$** (which is usually $n^k$)
2. The recursion cost: **$n^{\log_b a}$**

Compare these two polynomial bounds:

### Case 1: Recursion is heavier
If $n^{\log_b a}$ is strictly **greater** than $f(n)$ (i.e., $k < \log_b a$):
- **Complexity:** **$O(n^{\log_b a})$**

### Case 2: They are equally heavy
If $n^{\log_b a}$ is **equal** to $f(n)$ (i.e., $k = \log_b a$):
- You multiply the term by $\log n$.
- **Complexity:** **$O(n^{\log_b a} \cdot \log n)$**

### Case 3: External work is heavier
If $f(n)$ is strictly **greater** than $n^{\log_b a}$ (i.e., $k > \log_b a$):
- **Complexity:** **$O(f(n))$**

---

## 3. Exam Problem Walkthroughs

### Problem 1: Merge Sort
**Relation:** $T(n) = 2T(n/2) + n$
1. Extract variables: $a = 2, b = 2, f(n) = n^1$ (so $k=1$).
2. Calculate recursion cost: $n^{\log_b a} = n^{\log_2 2} = n^1 = n$.
3. Compare: $n^1$ vs $n^1$. They are **equal**.
4. Apply Case 2: Multiply by $\log n$.
**Answer:** $O(n \log n)$

### Problem 2: Binary Search
**Relation:** $T(n) = 1T(n/2) + c$
1. Extract variables: $a = 1, b = 2, f(n) = c = n^0$ (so $k=0$).
2. Calculate recursion cost: $n^{\log_b a} = n^{\log_2 1} = n^0 = 1$.
3. Compare: $1$ vs $1$. They are **equal**.
4. Apply Case 2: Multiply by $\log n$.
**Answer:** $O(\log n)$

### Problem 3: Karatsuba Multiplication
**Relation:** $T(n) = 3T(n/2) + n$
1. Extract variables: $a = 3, b = 2, f(n) = n^1$ (so $k=1$).
2. Calculate recursion cost: $n^{\log_b a} = n^{\log_2 3} \approx n^{1.58}$.
3. Compare: $n^{1.58}$ is **greater** than $n^1$.
4. Apply Case 1: The recursion cost dominates.
**Answer:** $O(n^{\log_2 3})$

### Problem 4
**Relation:** $T(n) = 2T(n/2) + n^2$
1. Extract variables: $a = 2, b = 2, f(n) = n^2$ (so $k=2$).
2. Calculate recursion cost: $n^{\log_b a} = n^{\log_2 2} = n^1 = n$.
3. Compare: $n^2$ is **greater** than $n^1$.
4. Apply Case 3: The external work $f(n)$ dominates.
**Answer:** $O(n^2)$

---

## 4. Exam Strategy & Practice Questions

### Typical Exam Questions:
1. **"State the Master Theorem and use it to solve $T(n) = 4T(n/2) + n$." (05 marks)**
   - *Strategy:* Write the generic formula $T(n) = aT(n/b) + f(n)$. State the 3 cases clearly as comparisons between $n^{\log_b a}$ and $f(n)$. Then solve the problem: $a=4, b=2, n^{\log_2 4} = n^2$. Since $n^2 > n$, Case 1 applies. Answer is $O(n^2)$.

### Practice Questions:
1. Use the Master Theorem to solve $T(n) = 8T(n/2) + n^2$. *(Answer: $n^{\log_2 8} = n^3$. $n^3 > n^2$. Case 1: $O(n^3)$).*
