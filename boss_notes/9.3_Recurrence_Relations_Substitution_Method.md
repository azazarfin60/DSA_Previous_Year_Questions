# 9.3 Recurrence Relations (Substitution Method)

> **Exam Frequency:** 🟡 High Frequency | **Typical Marks:** 04–06 | **Section:** B  
> **Course Outcome:** Maps to CO5

---

## 1. Core Definition
A **Recurrence Relation** is an equation that recursively defines a sequence or a function in terms of its previous terms. In algorithm analysis, they are used to determine the time complexity of recursive algorithms (like Merge Sort or Binary Search).

For example, a recursive function that halves the array and does $O(1)$ work can be expressed as:
$T(n) = T(n/2) + c$

---

## 2. The Substitution Method (Iteration Method)
The substitution method is a mathematical way to solve recurrence relations by repeatedly substituting the relation into itself until a pattern emerges, reducing the equation down to the base case $T(1)$.

### The 4-Step Process:
1. **Write the initial equation:** e.g., $T(n) = T(n/2) + c$.
2. **Substitute $T(n)$ multiple times:** Find $T(n/2)$, then $T(n/4)$, etc.
3. **Generalize the pattern:** Express the equation in terms of $k$ iterations.
4. **Apply the Base Case:** Set the generalized argument to the base case (usually $1$) to solve for $k$, then substitute $k$ back into the generalized equation to get the final Big-O complexity.

---

## 3. Exam Trace Example 1: Binary Search

**Solve: $T(n) = T(n/2) + c$**
*(Assume $T(1) = c$)*

**Step 1: Expand the relation**
- $T(n) = T(n/2) + c$  -- (Equation 1)
- What is $T(n/2)$? Substitute $n/2$ into the original formula:
  $T(n/2) = T(n/4) + c$

**Step 2: Substitute back into Eq 1**
- $T(n) = [T(n/4) + c] + c$
- $T(n) = T(n/4) + 2c$ -- (Equation 2)
- Substitute again: $T(n/4) = T(n/8) + c$
- $T(n) = [T(n/8) + c] + 2c$
- $T(n) = T(n/8) + 3c$ -- (Equation 3)

**Step 3: Generalize for 'k' iterations**
Notice the pattern: After $k$ steps, the formula is:
> **$T(n) = T(n / 2^k) + k \cdot c$**

**Step 4: Solve for $k$ using Base Case**
- We know the base case is $T(1)$. So, force the inner term to equal $1$:
- $n / 2^k = 1$
- $n = 2^k$
- Taking $\log$ on both sides: **$k = \log_2(n)$**

**Step 5: Final Substitution**
Substitute $k = \log_2(n)$ back into the generalized equation:
- $T(n) = T(1) + \log_2(n) \cdot c$
- $T(n) = c + c \cdot \log_2(n)$
- Drop the constants.

> **Final Complexity:** $O(\log n)$

---

## 4. Exam Trace Example 2: Merge Sort

**Solve: $T(n) = 2T(n/2) + n$**
*(Assume $T(1) = 1$)*

**Step 1: Expand the relation**
- $T(n) = 2T(n/2) + n$  -- (Eq 1)
- Find $T(n/2)$: $T(n/2) = 2T(n/4) + n/2$

**Step 2: Substitute back into Eq 1**
- $T(n) = 2[2T(n/4) + n/2] + n$
- $T(n) = 4T(n/4) + n + n$
- $T(n) = 4T(n/4) + 2n$ -- (Eq 2)
- Find $T(n/4)$: $T(n/4) = 2T(n/8) + n/4$
- $T(n) = 4[2T(n/8) + n/4] + 2n$
- $T(n) = 8T(n/8) + n + 2n$
- $T(n) = 8T(n/8) + 3n$ -- (Eq 3)

**Step 3: Generalize for 'k' iterations**
Notice the pattern ($4 = 2^2, 8 = 2^3$):
> **$T(n) = 2^k \cdot T(n / 2^k) + k \cdot n$**

**Step 4: Solve for $k$ using Base Case**
- Force inner term to equal $1$:
- $n / 2^k = 1$ $\implies n = 2^k \implies k = \log_2(n)$

**Step 5: Final Substitution**
Substitute $n$ for $2^k$, and $\log_2(n)$ for $k$:
- $T(n) = n \cdot T(1) + n \cdot \log_2(n)$
- $T(n) = n \cdot 1 + n \log n$
- As $n$ grows, $n \log n$ dominates $n$.

> **Final Complexity:** $O(n \log n)$

---

## 5. Exam Strategy & Practice Questions

### Typical Exam Questions:
1. **"Solve the recurrence relation $T(n) = 2T(n/2) + n$ using the substitution method." (05 marks)**
   - *Strategy:* This is the exact Merge Sort proof. Do not skip steps. Show exactly how the $n/2 \times 2 = n$ cancellation happens in Step 2. Clearly state the generalization to $k$, and the base case algebra $n = 2^k$.

### Practice Questions:
1. Solve $T(n) = T(n-1) + 1$ using substitution. *(Answer: $T(n) = O(n)$).*
