# 8.12 Spanning Tree

> **Exam Frequency:** 🟢 Moderate Frequency | **Typical Marks:** 02–03 | **Section:** B  
> **Course Outcome:** Maps to CO4

---

## 1. Core Definition
A **Spanning Tree** is a subgraph of an undirected, connected graph that includes **all the vertices** of the original graph with the **minimum possible number of edges**.

Because it is a tree, it cannot have any loops (cycles).

---

## 2. Key Properties
If a graph $G$ has $V$ vertices and $E$ edges:
1. Every spanning tree will have exactly **$V$ vertices**.
2. Every spanning tree will have exactly **$V - 1$ edges**.
3. A spanning tree has **no cycles** (no closed loops).
4. A spanning tree is **minimally connected** — removing even a single edge will disconnect the graph.
5. A spanning tree is **maximally acyclic** — adding even a single edge will instantly create a cycle.
6. A single graph can have **multiple different spanning trees**.

---

## 3. Mathematical Formula for Complete Graphs
If a graph is a **Complete Graph** (every vertex is connected to every other vertex), denoted as $K_n$, the total number of possible spanning trees it can form is given by **Cayley's Formula**:
> **Total Spanning Trees = $V^{(V-2)}$**

*Example:* A complete graph with 4 vertices ($K_4$) has $4^{(4-2)} = 4^2 = 16$ possible spanning trees.

---

## 4. Visual Example

**Original Graph (V=4, E=5):**
```text
  A --- B
  | \   |
  |  \  |
  C --- D
```

**Valid Spanning Tree 1 (V=4, E=3):**
```text
  A --- B
  |     
  |     
  C --- D
```

**Valid Spanning Tree 2 (V=4, E=3):**
```text
  A     B
   \    |
    \   |
  C --- D
```
*(Notice both have all 4 vertices, exactly 3 edges, and 0 cycles).*

---

## 5. Exam Strategy & Practice Questions

### Typical Exam Questions:
1. **"Define a Spanning Tree and list its properties." (03 marks)**
   - *Strategy:* Provide the 1-sentence definition. List exactly 3 bullet points: exactly $V$ vertices, exactly $V-1$ edges, and 0 cycles.
2. **"How many spanning trees can be formed from a complete graph of 5 vertices?" (02 marks)**
   - *Strategy:* State Cayley's formula $V^{(V-2)}$. Calculate $5^{(5-2)} = 5^3 = 125$.

### Practice Questions:
1. Draw three different spanning trees for a triangle graph with one extra edge dangling from a corner.
